# FormaUI v0.4 Release Notes

Date: 2026-04-26  
Status: GA  
Version Line: v0.4.1 -> v0.4.6

## TL;DR

FormaUI v0.4 将 v0.3 的“可组合扩展”升级为“可传播增长”，核心交付如下：

- 官方 landing route（`/landing`）上线，完成信息架构、组件化组合与 CTA 流程闭环。
- 交付 6 个营销场景 blocks，并提供 `marketing-launch` pack 进行场景化安装。
- 新增 landing 组件合规检查、SEO 基线、CTA 埋点验证、Lighthouse 门禁，形成发布级质量收口。

## Delivered Scope

### 1. Official Landing Experience

- 在 docs 应用内交付官方 landing 页面：`apps/docs/app/landing/page.tsx`。
- 按区段组件化拆分 Hero / Proof / CTA，并使用 `@formaui/blocks` 组合实现。
- 保持响应式布局与可访问性基础能力，CTA 入口覆盖 docs、quick start、examples、GitHub。

### 2. Marketing Blocks and Scenario Pack

新增并发布以下营销 blocks：

- `hero-cta`
- `feature-grid`
- `logo-cloud`
- `stats-strip`
- `faq-accordion`
- `final-cta`

配套交付：

- blocks 源码：`packages/blocks/src/*`
- registry items：`registry/blocks/*.json`
- 文档页：`apps/docs/app/blocks/*`
- 官方 pack：`marketing-launch`（`registry/packs/marketing-launch.json`）

### 3. Quality Gates and Release Safety

- Landing 组件合规脚本：`apps/docs/scripts/check-landing-component-usage.mjs`
- SEO 基线：`metadata` + `sitemap` + `robots`
- CTA 埋点事件覆盖：`cta_docs_click`、`cta_quick_start_click`、`cta_examples_click`、`cta_github_click`
- Lighthouse 门禁：Performance >= 90、Accessibility >= 95、Best Practices >= 95、SEO >= 95

## Validation Gates

Release gating commands:

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
pnpm registry:build
```

Landing quality checks:

```bash
pnpm --filter docs test
pnpm --filter docs lighthouse
```

Validation evidence snapshot:

- Registry totals: `36 components`, `17 blocks`, `3 templates`, `4 packs`, `2 themes`.
- Landing SEO、CTA analytics、component-compliance checks 均有自动化测试或脚本门禁。
- Lighthouse latest run (`pnpm --filter docs lighthouse`): performance `100`, accessibility `96`, best-practices `96`, seo `100`.

## Upgrade Notes (v0.3 -> v0.4)

1. 新增官方 landing 路径：`/landing`，并提供 `landing/architecture` 与 `landing/reuse-blocks` 文档页。
2. 推广页面可优先通过 `npx formaui pack add marketing-launch` 快速起步。
3. 建议将 docs 质量门禁纳入 CI：`pnpm --filter docs lighthouse` 与 landing 合规检查。

## Known Limitations

- `section_view` 曝光事件未纳入 v0.4 GA 范围，当前 GA 仅覆盖关键 CTA 事件埋点。
- 多语言 landing 与 A/B 文案实验仍属后续版本规划（P1）。
