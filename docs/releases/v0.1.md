# FormaUI v0.1

FormaUI v0.1 validates the source-owned design system workflow for React, TypeScript, Tailwind CSS, Radix UI, registry-driven installation, and SaaS / AI product blocks.

## Included

- Design tokens
- Default and avocado themes
- 13 primitive components
- 8 composite components
- 8 product blocks
- ai-console-lite template
- Registry schema and generated items
- CLI init/add/block/template/theme commands
- Docs site
- Next.js example app

## Limitations

- React only
- Next.js prioritized
- No private registry
- No Figma Kit
- No FormaUI Studio
