# FormaUI v0.1

- Release date: 2026-04-25
- Release tag: `v0.1`
- Tag target commit: `89e6273` (`docs(plan): mark v0.1 implementation tasks as completed` on branch `v0.1.6`)

FormaUI v0.1 is the first complete release of the source-owned design system workflow for React + TypeScript + Tailwind + Radix, including installable components, product blocks, template composition, registry generation, CLI installation flow, docs, and a working Next.js example.

## Highlights

- Source-owned architecture validated end-to-end.
- Design foundation shipped with typed tokens, two themes (`default` and `avocado`), and light/dark mode.
- Installable UI system shipped with 13 primitives and 8 composites.
- Product layer shipped with 8 SaaS/AI blocks and `ai-console-lite` template.
- Registry + CLI workflow shipped for `init`, `add`, `block add`, `template add`, and `theme add`.
- Documentation and example app shipped and production-buildable.

## Delivered Scope

### Design Foundation

- Typed token groups: `colors`, `radius`, `spacing`, `typography`, `shadow`, `motion`.
- Theme assets: `default.css`, `avocado.css` with light and dark selectors.
- Tailwind preset mapping semantic CSS variables into Tailwind theme extension.

### UI Components

- 13 primitives:
  - `Avatar`, `Badge`, `Button`, `Card`, `Checkbox`, `Dialog`, `DropdownMenu`, `Input`, `Select`, `Switch`, `Tabs`, `Textarea`, `Tooltip`
- 8 composites:
  - `AppSidebar`, `EmptyState`, `FormField`, `MetricCard`, `PageHeader`, `StatusBadge`, `ThemeSwitcher`, `UserMenu`

### Product Blocks and Template

- 8 required blocks:
  - `LoginPanel`, `PricingSection`, `DashboardShell`, `SettingsLayout`, `BillingPanel`, `ApiKeyManager`, `TokenUsageChart`, `AgentRunTimeline`
- Template:
  - `ai-console-lite` composed from dashboard + AI operation blocks/components.

### Registry and CLI

- Registry outputs generated:
  - 22 component items
  - 8 block items
  - 2 theme items
  - 1 template item
- CLI command set:
  - `formaui init`
  - `formaui add <name>`
  - `formaui block add <name>`
  - `formaui template add <name>`
  - `formaui theme add <name>`

### Docs and Example

- Docs app covers installation, quick start, design system, tokens, themes, components, blocks, CLI, registry, and template usage.
- Example Next.js app includes component showcase, blocks showcase, and `ai-console-lite` composition page.
