# FormaUI v0.9 Release Notes

Date: 2026-05-04
Status: Planned
Version Line: v0.9.x

## TL;DR

v0.9 upgrades `apps/web` from a basic brand site into a polished landing page for FormaUI's source-owned UI system. The release focuses on product storytelling, real UI surfaces, registry/CLI conversion paths, SEO metadata, and web quality gates.

## Planned Delivery Scope

- Editorial Product Gallery visual direction.
- New homepage narrative: Hero, Product Surfaces, Source Ownership, Build Flow, Scenario Gallery, Quality System, Proof, Final CTA, Footer.
- Updated v0.9 metadata, sitemap/robots tests, and homepage smoke coverage.
- Stable Docs/GitHub/Showcase/Blocks conversion paths.
- No changes to `apps/docs` infrastructure, registry protocol, or component package internals.

## Quality Gates

The GA release requires:

- `pnpm --filter web lint` -> PASS
- `pnpm --filter web test` -> PASS
- `pnpm --filter web build` -> PASS
- Desktop and mobile visual inspection at 375px, 768px, and 1440px.
