# FormaUI v0.7 Release Notes

Date: 2026-04-29  
Status: Draft  
Version Line: v0.7.1 -> v0.7.6

## TL;DR

v0.7.1 冻结了 16 个基础组件的职责边界、最小 API 与依赖策略，并建立了 v0.7 交付矩阵占位。

## Contract Freeze (v0.7.1)

### Scope and Responsibility Boundaries

- `tree-select` 聚焦层级选择，不替代 `select` 的平铺场景。
- `cascader` 聚焦路径式逐级选择，不替代 `tree-select` 的树形直选。
- `segmented` 聚焦轻量模式切换，不替代 `tabs` 的内容分区容器。
- `anchor` / `affix` / `backtop` 分别覆盖页内导航、固定定位、回顶动作。

### Dependency Strategy

- v0.7 P0 仅 `segmented` 使用 Radix（`@radix-ui/react-toggle-group`）；其他组件优先 non-Radix 组合实现。
- `tree` / `cascader` / `tree-select` / `transfer` 禁止引入重型第三方树组件依赖。
- 新组件统一受控约定：`value` / `onValueChange` 优先，按需补充 `defaultValue`。
