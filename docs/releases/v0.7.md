## Release Scope Summary

- 组件能力：新增 16 个基础组件，覆盖层级选择、导航定位、反馈展示、媒体渲染。
- Registry/CLI：新增组件可通过 `list/search/info/add` 被检索与安装，且 metadata 完整。
- 文档与示例：16 个组件文档路由可达，`examples/next-app` 增加 v0.7 组合场景验证。

## Migration Notes

- 无对外破坏性 API 变更。
- 若在 Next.js App Router 中直接消费带 Radix/交互上下文的页面组合，建议显式使用客户端边界（`"use client"`）组织页面或子组件。
