# FormaUI v0.8 Release Notes

Date: 2026-05-01  
Status: GA Ready  
Version Line: v0.8.1 -> v0.8.5

## TL;DR

v0.8 完成文档服务升级：采用 Fumadocs 原生内容架构，文档信息架构对齐 benchmark 栏目模型，搜索与 SEO 基线接入，并通过全仓质量门禁。

## Delivery Timeline

| Wave | Scope | Status |
| --- | --- | --- |
| v0.8.1 | Docs core bootstrap (`content/docs` + `/docs/[...slug]`) | done |
| v0.8.2 | Core IA migration (Getting Started / Foundations / Components / Guides / CLI / Registry) | done |
| v0.8.3 | Full content migration (Blocks / Templates / Resources / Migration / Release Notes) | done |
| v0.8.4 | Search + SEO baseline (Orama API / sitemap / robots / IA tests) | done |
| v0.8.5 | GA hardening (governance guides + full quality gates + GA checklist) | done |

## Migration Scope Summary

Canonical docs content now lives under:

- `apps/docs/content/docs/getting-started/**`
- `apps/docs/content/docs/foundations/**`
- `apps/docs/content/docs/components/**`
- `apps/docs/content/docs/blocks/**`
- `apps/docs/content/docs/templates/**`
- `apps/docs/content/docs/guides/**`
- `apps/docs/content/docs/cli/**`
- `apps/docs/content/docs/registry/**`
- `apps/docs/content/docs/migration/**`
- `apps/docs/content/docs/release-notes/**`
- `apps/docs/content/docs/resources/**`

Navigation is source-driven by `meta.json` at root and domain levels.

## Search and SEO Baseline

- Search API: `apps/docs/app/api/search/route.ts` (`createFromSource(source)`)
- Root provider search endpoint: `/api/search`
- Sitemap expanded to cover `/docs/*` core domains
- Robots allowlist updated for docs domains and keeps `/api/` + `/_next/` restricted

## Docs Governance Additions

- `apps/docs/content/docs/guides/contributing-docs.mdx`
- `apps/docs/content/docs/guides/content-style-guide.mdx`

Manual sample validation for “15-minute new page flow” was completed in v0.8.5 by adding these two pages, updating `guides/meta.json`, and passing docs gates (`test/lint/build`).

## Known Limitations

- 部分历史 `apps/docs/app/*` 路由页仍存在（用于兼容当前仓库内容组织），v0.8 的 canonical 文档来源已切换为 `content/docs`。
- 搜索召回质量目前以结构正确性与关键词抽样为主，后续版本继续优化相关性和召回排序。

## GA Ready Conclusion

v0.8 达到 GA 条件，可进入 tag/发布流程。

Residual risks:

1. 外部深链仍可能引用历史非 `/docs/*` 路径，需在后续版本继续治理引用面。
