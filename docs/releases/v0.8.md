# FormaUI v0.8 Release Notes

Date: 2026-04-30  
Status: In Progress  
Version Line: v0.8.1 -> v0.8.3

## TL;DR

v0.8 聚焦文档服务升级：采用 Fumadocs 原生架构，统一 `/docs/*` 路由，并将文档信息架构对齐 benchmark 栏目模型。

## Scope Delivered So Far

- v0.8.1: Docs core bootstrap (`content/docs` source pipeline + `/docs/[...slug]`).
- v0.8.2: Core IA migration (Getting Started / Foundations / Components / Guides / CLI / Registry).
- v0.8.3: Full domain migration for Blocks / Templates / Resources / Migration / Release Notes.

## IA Snapshot (v0.8)

Top-level docs domains:

- Getting Started
- Foundations
- Components
- Blocks
- Templates
- Guides
- CLI
- Registry
- Migration
- Release Notes
- Resources

## Migration Notes

- Legacy app-route docs pages are being migrated to `apps/docs/content/docs/**` as canonical source.
- Ordering is controlled via per-domain `meta.json`.

## Quality Gates (Current)

At this stage, docs-level gates are expected to pass:

- `pnpm --filter docs test`
- `pnpm --filter docs lint`
- `pnpm --filter docs build`

Full-repo GA gates and final evidence will be completed in v0.8.5.
