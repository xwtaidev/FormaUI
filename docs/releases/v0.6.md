# FormaUI v0.6 Release Notes

Date: 2026-04-26  
Status: Draft  
Version Line: v0.6.1 -> v0.6.7

## TL;DR

v0.6.1 冻结了 20 个基础组件的命名、别名、最小 API 与依赖策略，并建立了后续分波次交付的发布矩阵占位。

## Contract Freeze (v0.6.1)

### Official Naming and Alias Policy

- Official: `drawer`, Alias (compat-only): `sheet`
- Official: `combobox`, Alias (compat-only): `autocomplete`
- 别名不作为独立组件条目，不进入 docs 导航主列表

### Radix Dependency Baseline

- `@radix-ui/react-collapsible`
- `@radix-ui/react-context-menu`
- `@radix-ui/react-label`
- `@radix-ui/react-menubar`
