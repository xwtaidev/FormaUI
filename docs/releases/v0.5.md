# FormaUI v0.5 Release Notes

Date: 2026-04-26  
Status: RC (v0.5.5 quality gates passed; v0.5.6 pending GA checklist)

## TL;DR

FormaUI v0.5 完成了官网与文档站拆分：`apps/web` 聚焦品牌与增长叙事，`apps/docs` 聚焦开发者文档与迁移指南。  
在 v0.5.5 中，已完成跨站回归测试扩展与全仓质量门禁验证，为 v0.5.6 GA 人工核对提供稳定基线。
