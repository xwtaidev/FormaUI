# Design System

FormaUI v0.1 is organized into these layers:

1. Tokens
2. Themes
3. Primitive components
4. Composite components
5. Product blocks
6. Templates
7. Registry and CLI

## Source-Owned Workflow

1. Pick an install target with CLI.
2. Resolve registry dependencies.
3. Copy source files into your project.
4. Continue product development in your own repository.
