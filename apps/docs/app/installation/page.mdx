# Installation

Use a React + TypeScript project (Next.js is recommended for v0.1).

## Prerequisites

- Node.js 20+
- npm, pnpm, yarn, or bun
- Existing React project with a CSS entry file

## Install CLI

```bash
npm install -g formaui
```

Or run with npx:

```bash
npx formaui init
```

## Initialize FormaUI

```bash
npx formaui init
```

This writes:

- `formaui.json`
- `lib/utils.ts`
- `styles/formaui.css`
