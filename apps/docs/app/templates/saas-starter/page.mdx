# saas-starter

## Overview

`saas-starter` is a production-oriented template that combines pricing, operations visibility, and model control in one starter route.

## Install Command

```bash
npx formaui template add saas-starter
```

## What It Installs

- `app/saas-starter/page.tsx`
- Template registry metadata and all required registry dependencies

## Composition

Primary assets included:

- `pricing-section`
- `team-members-table`
- `notification-panel`
- `model-selector`
- `page-header`
- `metric-card`
- `button`
- `badge`
