# ai-console-lite

## Overview

`ai-console-lite` is a template composition that combines dashboard structure, metrics, charts, timelines, settings, and key management.

## Install Command

```bash
npx formaui template add ai-console-lite
```

## What It Installs

- `app/ai-console-lite/page.tsx`
- Registry dependencies required by the template graph

## Composition

The template composes these primary assets:

- `dashboard-shell`
- `token-usage-chart`
- `agent-run-timeline`
- `api-key-manager`
- `settings-layout`
- `page-header`
- `metric-card`
- `button`
