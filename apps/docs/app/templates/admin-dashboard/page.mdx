# admin-dashboard

## Overview

`admin-dashboard` is an operations control template for usage visibility, run timelines, incident response, and model governance.

## Install Command

```bash
npx formaui template add admin-dashboard
```

## What It Installs

- `app/admin-dashboard/page.tsx`
- Template registry metadata and all required registry dependencies

## Composition

Primary assets included:

- `dashboard-shell`
- `token-usage-chart`
- `agent-run-timeline`
- `team-members-table`
- `notification-panel`
- `model-selector`
- `page-header`
- `metric-card`
- `button`
