# Quick Start

Follow this path to compose a product surface quickly.

```bash
npx formaui init
npx formaui add button
npx formaui block add dashboard-shell
npx formaui template add ai-console-lite
npx formaui theme add avocado
```

## Result

- You own the copied source code inside your project
- Registry dependencies are installed in dependency order
- You can customize structure, copy, and styles directly
