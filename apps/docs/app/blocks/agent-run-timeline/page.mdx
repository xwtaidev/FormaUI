# AgentRunTimeline

## Preview

Timeline list for agent run status, duration, and recent activity context.

## Install Command

```bash
npx formaui block add agent-run-timeline
```

## Use Case

Audit model or agent execution history in console-style interfaces.

## Dependencies

Registry dependencies:

- `card`
- `status-badge`

## Responsive Behavior

Behaves as a stacked list component and adapts to both side-by-side and full-width layouts.

## Customization Guide

Replace demo runs with backend event data and map statuses to your own operational taxonomy.
