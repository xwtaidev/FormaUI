# NotificationPanel

## Overview

`NotificationPanel` is an operations block for event triage, alert visibility, and quick actions.

## Install Command

```bash
npx formaui block add notification-panel
```

## Included Composition

- Alert cards (`card`)
- Severity tags (`badge`)
- Action buttons (`button`)
- Quick actions (`search-command`)

## Responsive Behavior

Each notification item stacks on mobile and switches to row alignment on larger viewports.
