# TokenUsageChart

## Preview

A card-style chart section for token usage trend visibility.

## Install Command

```bash
npx formaui block add token-usage-chart
```

## Use Case

Track model token consumption by day or interval in AI product dashboards.

## Dependencies

Registry dependencies:

- `card`

## Responsive Behavior

Fits a single grid cell and scales naturally when placed in one or two column dashboard sections.

## Customization Guide

Swap demo data points with your product telemetry and tune labels or axis density for your scale.
