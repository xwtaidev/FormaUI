# Theme

FormaUI v0.1 themes are CSS-variable based.

## Built-in Themes

- `default`
- `avocado`

Each theme includes both light and dark variable sets.

## Install Commands

```bash
npx formaui theme add default
npx formaui theme add avocado
```

## Customize

You can override variables in your project CSS after install, for example `--primary`, `--background`, and `--radius`.
