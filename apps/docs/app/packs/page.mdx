# Packs

## Overview

Packs install a curated set of components/blocks for a scenario so teams can start from a usable baseline instead of
adding assets one-by-one.

## Discover Packs

```bash
npx formaui list --kind pack
npx formaui pack list
npx formaui pack list --category dashboard
npx formaui pack list --scenario loading
```

## Inspect and Install

```bash
npx formaui pack info dashboard-foundation
npx formaui pack add dashboard-foundation --dry-run
npx formaui pack add dashboard-foundation
npx formaui pack info marketing-launch
npx formaui pack add marketing-launch --dry-run
npx formaui pack add marketing-launch
```

## Official Packs

- [dashboard-foundation](/packs/dashboard-foundation): table + filter + pagination + empty-result recovery
- [data-entry](/packs/data-entry): input/status/date form flows
- [feedback-loading](/packs/feedback-loading): loading, progress, status, and empty-state feedback
- [marketing-launch](/packs/marketing-launch): hero + feature grid + logo cloud + stats + faq + final cta

## Notes

- `pack add` resolves every `registryDependencies` entry recursively.
- Use `--dry-run` first in CI or during migrations.
- Use `--category` and `--scenario` filters to find the right pack faster.
