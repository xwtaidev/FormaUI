# marketing-launch

## Overview

`marketing-launch` bundles the official FormaUI marketing blocks so teams can assemble a landing page baseline quickly.

## Install Command

```bash
npx formaui pack add marketing-launch
```

## Dry Run

```bash
npx formaui pack add marketing-launch --dry-run
```

## Included Registry Dependencies

- `hero-cta`
- `feature-grid`
- `logo-cloud`
- `stats-strip`
- `faq-accordion`
- `final-cta`

## Typical Use

- Build an official landing in docs or product marketing surfaces
- Start from shared blocks before writing product-specific variants
- Keep section composition aligned with FormaUI tokens and components
