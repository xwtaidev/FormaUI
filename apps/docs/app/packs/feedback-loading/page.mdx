# feedback-loading

## Overview

`feedback-loading` bundles loading and status feedback primitives for async flows, empty states, and recovery actions.

## Install Command

```bash
npx formaui pack add feedback-loading
```

Dry-run before writing files:

```bash
npx formaui pack add feedback-loading --dry-run
```

## Best-fit Scenarios

- Data fetching transitions and skeleton placeholders
- Empty-result and retry experiences
- Status communication inside dialogs and tooltips
