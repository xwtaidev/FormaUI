# dashboard-foundation

## Overview

`dashboard-foundation` is the baseline pack for analytics/operations dashboards with search, filtering, pagination,
and empty-result handling.

## Install Command

```bash
npx formaui pack add dashboard-foundation
```

Dry-run before writing files:

```bash
npx formaui pack add dashboard-foundation --dry-run
```

## Best-fit Scenarios

- Admin and operations dashboards
- Team or tenant directories with table search
- KPI pages with fallback states for no-match queries

## What It Installs

- `dashboard-shell`
- `page-header`
- `metric-card`
- `data-table`
- `data-table-toolbar`
- `filter-bar`
- `pagination-bar`
- `empty-search-state`
