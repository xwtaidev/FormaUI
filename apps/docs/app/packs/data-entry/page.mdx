# data-entry

## Overview

`data-entry` is the starter pack for onboarding/settings forms that need input, status selection, and date-bound
filter controls.

## Install Command

```bash
npx formaui pack add data-entry
```

Dry-run before writing files:

```bash
npx formaui pack add data-entry --dry-run
```

## Best-fit Scenarios

- Workspace settings and onboarding forms
- Backoffice CRUD pages with status/date filters
- Multi-step forms that need shared input primitives

## What It Installs

- `form-field`
- `input`
- `select`
- `radio-group`
- `checkbox`
- `date-range-picker`
- `filter-bar`
- `button`
- `card`
