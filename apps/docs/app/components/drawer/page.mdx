# Drawer

## Overview

`Drawer` presents side-mounted overlay panels for settings, previews, and multi-step side tasks.

## Installation

```bash
npx formaui add drawer
```

## Usage

```tsx
import {
  Drawer,
  DrawerContent,
  DrawerDescription,
  DrawerHeader,
  DrawerTitle,
  DrawerTrigger
} from "@/components/primitives/drawer";

export function SettingsDrawer() {
  return (
    <Drawer>
      <DrawerTrigger className="rounded-md border px-3 py-2 text-sm">Open settings</DrawerTrigger>
      <DrawerContent side="right">
        <DrawerHeader>
          <DrawerTitle>Workspace Settings</DrawerTitle>
          <DrawerDescription>Update notification, privacy, and access preferences.</DrawerDescription>
        </DrawerHeader>
      </DrawerContent>
    </Drawer>
  );
}
```

## Accessibility

- Built on Radix dialog primitives for focus lock and escape-close behavior.
- Supports directional placement with the `side` prop.
- Includes labeled title/description semantics for screen readers.
