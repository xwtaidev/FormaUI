# Alert

## Overview

`Alert` is a semantic feedback container for inline warnings, errors, and operational notices.

## Installation

```bash
npx formaui add alert
```

## Usage

```tsx
import { Alert } from "@/components/primitives/alert";

export function BillingAlert() {
  return (
    <Alert
      variant="warning"
      title="Usage threshold reached"
      description="Your project consumed 85% of this month's credits."
    />
  );
}
```

## Accessibility

- Defaults to `role="status"` for assistive technologies.
- Supports clear title/description hierarchy for fast screen-reader parsing.
- Works well with keyboard-focusable actions when composed with `button` or `link`.
