# Timeline

## Overview

`Timeline` presents chronological events such as release milestones, approval chains, and activity logs.

## Installation

```bash
npx formaui add timeline
```

## Usage

```tsx
import { Timeline } from "@/components/primitives/timeline";

const items = [
  { key: "scope", label: "Scope", children: "v0.7.1 contract freeze" },
  { key: "wave-a", label: "Wave A", children: "Input and selection completed" }
];

export function ReleaseTimeline() {
  return <Timeline items={items} />;
}
```

## v0.7 Scope

- Event list with label + content blocks.
- Supports pending tail node.
- Does not include virtualized long-list behavior.
