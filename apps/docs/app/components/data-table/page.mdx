# DataTable

## Overview

`DataTable` is a composite for sortable tabular views with custom cell rendering and empty-state fallback.

## Install Command

```bash
npx formaui add data-table
```

## Usage

```tsx
import { DataTable } from "@/components/composites/data-table";

const rows = [
  { name: "Avery Lin", role: "Product Lead", score: 98 },
  { name: "Riley Chen", role: "ML Engineer", score: 92 }
];

export function TeamTable() {
  return (
    <DataTable
      columns={[
        { key: "name", header: "Name", sortable: true },
        { key: "role", header: "Role" },
        { key: "score", header: "Score", sortable: true, align: "right" }
      ]}
      rows={rows}
    />
  );
}
```

## Key Features

- Sort toggle by column (`asc`/`desc`)
- Custom cell renderer for badges/actions
- Alignment controls per column
