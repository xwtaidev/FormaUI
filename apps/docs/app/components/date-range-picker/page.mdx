# DateRangePicker

## Overview

`DateRangePicker` offers a lightweight start/end date selection panel for reporting windows and audit history filters.

## Install Command

```bash
npx formaui add date-range-picker
```

## Usage

```tsx
import { DateRangePicker } from "@/components/composites/date-range-picker";

export function ReportRange() {
  return (
    <DateRangePicker
      value={{ from: "2026-04-01", to: "2026-04-30" }}
      onChange={(range) => console.log(range)}
    />
  );
}
```

## Scenario Guidance

- Use in analytics/reporting screens to scope metrics by release period.
- Keep labels explicit (`Start date`/`End date`) to reduce date ambiguity.
- Pair with status/search controls when users need multi-dimensional filtering.
