# Segmented

## Overview

`Segmented` provides compact mode switching for tightly-related options in one row.

## Installation

```bash
npx formaui add segmented
```

## Usage

```tsx
import { Segmented } from "@/components/primitives/segmented";

export function FilterModeSwitch() {
  return (
    <Segmented
      options={[
        { label: "All", value: "all" },
        { label: "Open", value: "open" },
        { label: "Closed", value: "closed" }
      ]}
      defaultValue="all"
    />
  );
}
```

## Scope Boundary (v0.7)

- Use `segmented` for lightweight state mode toggles.
- Use `tabs` when each option maps to a full content panel region.
- Use `radio-group` when form semantics and vertical/field layouts are the primary concern.
