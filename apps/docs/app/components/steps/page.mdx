# Steps

## Overview

`Steps` visualizes multi-stage workflows such as onboarding, setup, or release pipelines.

## Installation

```bash
npx formaui add steps
```

## Usage

```tsx
import { Steps, StepsItem } from "@/components/primitives/steps";

export function ReleaseSteps() {
  return (
    <Steps current={1} aria-label="Release checklist">
      <StepsItem index={0} heading="Freeze contracts" description="Finalize naming and API scope." />
      <StepsItem index={1} heading="Ship Wave A" description="Deliver alert, breadcrumb, label, typography, and steps." />
      <StepsItem index={2} heading="Docs convergence" description="Publish docs and registry entries." />
    </Steps>
  );
}
```

## Accessibility

- Supports keyboard traversal with `Arrow` keys, `Home`, and `End`.
- Active step is exposed through `aria-current="step"`.
- Works in horizontal and vertical layouts via the `direction` prop.
