# Introduction

FormaUI is a source-owned design system for modern SaaS and AI product interfaces.

Unlike traditional npm-only component libraries, FormaUI copies source files into your project so your team can edit, version, and evolve every component, block, and template.

## What v0.1 Includes

- Design tokens and CSS variable themes (`default`, `avocado`) with light and dark support
- 13 primitive components and 8 composite components
- 8 SaaS and AI product blocks
- Registry item schema and generated registry JSON
- CLI commands for `init`, `add`, `block add`, `template add`, and `theme add`
- Next.js example app and complete docs site
