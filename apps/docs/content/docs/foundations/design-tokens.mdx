# Design Tokens

v0.1 ships token categories for spacing, radius, typography, shadow, color, and motion.

## Token Forms

- TypeScript exports in `packages/tokens/src/*`
- CSS variables consumed by themes
- Tailwind mappings through `@formaui/tailwind-preset`

## Why This Matters

Tokens keep components and blocks visually consistent while still letting product teams customize brand direction later.
