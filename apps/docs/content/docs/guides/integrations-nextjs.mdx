# Integrations for Next.js

## Install

```bash
pnpm add @formaui/components @formaui/blocks
```

## Initialize CLI

```bash
npx formaui init
```

## Recommended Setup

1. Ensure `react` and `next` are already installed.
2. Configure Tailwind and global CSS entry.
3. Add required alias paths in your TS config.
4. Install primitives/blocks with `formaui add` and `formaui block add`.

## Best Practices

- Use `--dry-run` before batch installs.
- Keep installed source files editable in your repository.
