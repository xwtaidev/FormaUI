# Integrations for Vite

## Install

```bash
pnpm add @formaui/components @formaui/blocks
```

## Initialize CLI

```bash
npx formaui init
```

## Recommended Setup

1. Start from a React + TypeScript Vite template.
2. Enable Tailwind and include FormaUI styles.
3. Configure import aliases (`@/components`, `@/lib`, `@/styles`).
4. Install components/blocks from the registry.

## Best Practices

- Keep generated files source-owned and version-controlled.
- Prefer pack commands for scenario-based adoption.
