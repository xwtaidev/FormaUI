# Commands

本页提供 FormaUI CLI 的参数级参考，适合作为日常查询手册。

## 全局参数

| 参数 | 说明 |
| --- | --- |
| `--cwd <path>` | 在目标项目目录执行命令 |
| `--registry <source>` | 指定远端或本地 registry 源（URL 或本地路径） |
| `--dry-run` | 仅输出安装计划，不写入文件 |
| `-y, --yes` | 覆盖冲突文件时跳过确认 |

## `formaui init`

初始化项目配置与基础文件。

```bash title="Usage"
npx formaui init
npx formaui init --cwd apps/web
npx formaui init --yes
```

## `formaui list`

列出可安装资产。

```bash title="Usage"
npx formaui list
npx formaui list --kind component
npx formaui list --kind pack
npx formaui list --category dashboard
npx formaui list --scenario analytics
```

## `formaui search <query>`

按关键词检索资产名称与元数据。

```bash title="Usage"
npx formaui search chart
npx formaui search dashboard --kind block
npx formaui search table --scenario dashboard
```

## `formaui info <name>`

查看单个资产详情（类型、依赖、文件写入目标、稳定性等）。

```bash title="Usage"
npx formaui info data-table
npx formaui info dashboard-shell --kind block
npx formaui info ai-console-lite --kind template
```

## `formaui add <name>`

安装通用资产（由 CLI 根据名称和 registry 数据解析）。

```bash title="Usage"
npx formaui add button
npx formaui add feature-grid --dry-run
npx formaui add data-table --cwd apps/docs
```

## `formaui block add <name>`

显式安装 block 类型资产。

```bash title="Usage"
npx formaui block add dashboard-shell
npx formaui block add token-usage-chart --dry-run
```

## `formaui template add <name>`

安装 template 及其依赖图。

```bash title="Usage"
npx formaui template add ai-console-lite
npx formaui template add saas-starter --dry-run
```

## `formaui theme add <name>`

安装主题资产。

```bash title="Usage"
npx formaui theme add default
npx formaui theme add avocado --cwd apps/web
```

## `formaui pack <list|info|add>`

批量安装场景包。

```bash title="Usage"
npx formaui pack list
npx formaui pack list --category forms
npx formaui pack info dashboard-foundation
npx formaui pack add dashboard-foundation
npx formaui pack add marketing-launch --dry-run
```

## `formaui doctor`

检查项目环境与配置健康度。

```bash title="Usage"
npx formaui doctor
npx formaui doctor --cwd apps/web
```

## 常见命令组合

### 新项目首轮接入

```bash title="Bootstrap"
npx formaui init
npx formaui list --kind component
npx formaui add button
npx formaui doctor
```

### 先发现再安装

```bash title="Discover then Install"
npx formaui search table --scenario dashboard
npx formaui info data-table --kind component
npx formaui add data-table --dry-run
npx formaui add data-table
```

### Monorepo 定向安装

```bash title="Monorepo"
npx formaui add feature-grid --cwd apps/web
npx formaui template add ai-console-lite --cwd apps/web --dry-run
npx formaui doctor --cwd apps/web
```

## 故障排查

### 命令执行成功但文件不在预期目录

优先检查是否传入 `--cwd`，并用 `--dry-run` 预览写入计划。

### 资产名称冲突或歧义

先执行 `info <name> --kind <type>`，确认类型后再安装。

### 依赖安装失败

检查包管理器锁文件、网络源与权限，然后重试命令。
