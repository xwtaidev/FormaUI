# Figma

## Design Kit

- Use the FormaUI component and token definitions as the source of truth for UI primitives.
- Keep variant names aligned with code component names to reduce translation costs.

## Design-to-Code Workflow

1. Start from component and block definitions in docs.
2. Align naming and states in Figma components.
3. Validate behavior and accessibility constraints in code before publishing.
