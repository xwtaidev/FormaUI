# Examples

## Official Example Apps

- `examples/next-app`: Next.js baseline integration.
- `examples/vite-app`: Vite + React baseline integration.

## Recommended Workflow

1. Start with the example closest to your runtime.
2. Install required components/blocks/templates with CLI.
3. Move shared abstractions into your own domain packages.
