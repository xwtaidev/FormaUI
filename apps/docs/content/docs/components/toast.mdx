# Toast

## Overview

`Toast` provides transient feedback messages with Radix-powered viewport, close action, and variant styling.

## Installation

```bash
npx formaui add toast
```

## Usage

```tsx
import { Toast, ToastClose, ToastDescription, ToastProvider, ToastTitle, ToastViewport } from "@/components/primitives/toast";

export function SaveToast() {
  return (
    <ToastProvider>
      <Toast open>
        <div>
          <ToastTitle>Saved</ToastTitle>
          <ToastDescription>All changes synced.</ToastDescription>
        </div>
        <ToastClose />
      </Toast>
      <ToastViewport />
    </ToastProvider>
  );
}
```

## Accessibility

- Uses Radix toast semantics with a dedicated notification region.
- Close action is keyboard-focusable and includes a screen-reader label.
- Supports variant styling (`default` / `destructive`) while preserving readable contrast.
