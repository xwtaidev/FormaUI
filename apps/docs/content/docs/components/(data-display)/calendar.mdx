# Calendar

`Calendar` 提供月视图日期选择能力，支持月份切换、单日期选中与受控/非受控模式。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[340px] items-center justify-center p-6">
    <CalendarInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/data-display-demos.tsx"
import * as React from "react";
import { Calendar } from "@formaui/components";

export function BookingCalendar() {
  const [date, setDate] = React.useState<Date | undefined>(new Date());

  return <Calendar value={date} onSelect={setDate} />;
}
```

## API Reference

### `Calendar` Props

| Prop           | 类型                   | 默认值 | 说明               |
| -------------- | ---------------------- | ------ | ------------------ |
| `value`        | `Date`                 | `-`    | 受控选中日期       |
| `defaultValue` | `Date`                 | `-`    | 非受控初始选中日期 |
| `onSelect`     | `(date: Date) => void` | `-`    | 日期选中回调       |
| `className`    | `string`               | `-`    | 追加容器样式       |

### 继承属性

`Calendar` 继承 `React.HTMLAttributes<HTMLDivElement>`（排除 `onSelect/value/defaultValue`），可直接使用：

- `id`
- `aria-*`
- `data-*`
- `onMouseEnter`

## 可访问性

- Day cells use full date labels (for example `April 20, 2026`).
- Current selection is exposed through `data-selected` for styling and state checks.
- Month navigation controls include explicit `Previous month` and `Next month` labels.
