# Steps

`Steps` 用于展示多阶段流程，支持当前步骤高亮、完成状态与键盘导航。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[280px] items-center justify-center p-6">
    <StepsInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/data-display-demos.tsx"
import { Steps, StepsItem } from "@formaui/components";

export function ReleaseSteps() {
  return (
    <Steps current={1} aria-label="Release checklist">
      <StepsItem
        index={0}
        heading="Freeze contracts"
        description="Finalize naming and API scope."
      />
      <StepsItem
        index={1}
        heading="Ship Wave A"
        description="Deliver docs and component updates."
      />
      <StepsItem
        index={2}
        heading="Publish release"
        description="Sync changelog and release tags."
      />
    </Steps>
  );
}
```

## API Reference

### `Steps` Props

| Prop        | 类型                         | 默认值         | 说明         |
| ----------- | ---------------------------- | -------------- | ------------ |
| `current`   | `number`                     | `-`            | 当前步骤索引 |
| `direction` | `"horizontal" \| "vertical"` | `"horizontal"` | 步骤排列方向 |
| `className` | `string`                     | `-`            | 追加容器样式 |

### `StepsItem` Props

| Prop          | 类型                                   | 默认值   | 说明           |
| ------------- | -------------------------------------- | -------- | -------------- |
| `index`       | `number`                               | `-`      | 步骤索引       |
| `heading`     | `React.ReactNode`                      | `-`      | 步骤标题       |
| `description` | `React.ReactNode`                      | `-`      | 步骤描述       |
| `status`      | `"pending" \| "active" \| "completed"` | 自动推导 | 步骤状态       |
| `disabled`    | `boolean`                              | `false`  | 禁用当前步骤项 |
| `className`   | `string`                               | `-`      | 追加步骤项样式 |

## 可访问性

- Supports keyboard traversal with `Arrow` keys, `Home`, and `End`.
- Active step is exposed through `aria-current="step"`.
- Works in horizontal and vertical layouts via the `direction` prop.
