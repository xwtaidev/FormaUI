# Progress

`Progress` 用于表达阶段完成比例，支持受控数值与可访问进度语义。

```tsx title="components/demos/data-display-demos.tsx"
import { Progress } from "@formaui/components";

export function UploadProgress() {
  return (
    <div className="space-y-2">
      <p className="text-sm text-muted-foreground">Uploading files...</p>
```

    </div>

## API Reference

### `Progress` Props

`Progress` 继承 `React.ComponentPropsWithoutRef<typeof ProgressPrimitive.Root>`：

| Prop        | 类型     | 默认值 | 说明                           |
| ----------- | -------- | ------ | ------------------------------ |
| `value`     | `number` | `0`    | 当前进度值（会被归一到 0-100） |
| `max`       | `number` | `100`  | 最大值（Radix 默认）           |
| `className` | `string` | `-`    | 追加容器样式                   |

## 可访问性

- Exposes `role="progressbar"` and `aria-valuenow` through Radix.
- Accepts controlled `value` in the 0-100 range.
- Works well with screen-reader friendly status text nearby.
