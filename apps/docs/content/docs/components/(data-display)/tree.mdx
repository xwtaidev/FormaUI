# Tree

`Tree` 用于展示层级数据，支持节点展开/收起、选中状态与受控/非受控管理。

    <TreeInteractiveDemo />

</div>

```tsx title="components/demos/data-display-demos.tsx"
import { Tree } from "@formaui/components";

const treeData = [
  {
    key: "engineering",
    title: "Engineering",
    children: [
      { key: "frontend", title: "Frontend" },
      { key: "backend", title: "Backend" },
    ],
  },
];

export function TeamTree() {
  return <Tree data={treeData} defaultExpandedKeys={["engineering"]} />;
}
```

## API Reference

### `Tree` Props

| Prop                  | 类型                               | 默认值  | 说明               |
| --------------------- | ---------------------------------- | ------- | ------------------ |
| `data`                | `TreeNode[]`                       | `-`     | 树节点数据（必填） |
| `expandedKeys`        | `string[]`                         | `-`     | 受控展开节点       |
| `defaultExpandedKeys` | `string[]`                         | `[]`    | 非受控初始展开节点 |
| `selectedKeys`        | `string[]`                         | `-`     | 受控选中节点       |
| `defaultSelectedKeys` | `string[]`                         | `[]`    | 非受控初始选中节点 |
| `onExpandedChange`    | `(expandedKeys: string[]) => void` | `-`     | 展开状态变化回调   |
| `onSelectedChange`    | `(selectedKeys: string[]) => void` | `-`     | 选中状态变化回调   |
| `multiple`            | `boolean`                          | `false` | 是否允许多选       |
| `className`           | `string`                           | `-`     | 追加容器样式       |

### `TreeNode` 结构

| 字段       | 类型              | 默认值 | 说明           |
| ---------- | ----------------- | ------ | -------------- |
| `key`      | `string`          | `-`    | 节点唯一键值   |
| `title`    | `React.ReactNode` | `-`    | 节点标题       |
| `disabled` | `boolean`         | `-`    | 禁用选中与交互 |
| `children` | `TreeNode[]`      | `-`    | 子节点列表     |

## 可访问性

- 节点按钮提供 `aria-expanded` 与 `aria-selected` 状态语义。
- 默认支持键盘 `ArrowLeft/ArrowRight/Enter/Space` 操作。
- 层级深度建议保持可读，超长树结构可分段或按需加载。
