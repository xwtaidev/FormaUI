# Avatar

`Avatar` 用于展示用户头像与身份占位信息，支持图片加载失败后的降级回退。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[220px] items-center justify-center p-6">
    <AvatarInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/data-display-demos.tsx"
import { Avatar, AvatarFallback, AvatarImage } from "@formaui/components";

export function UserAvatar() {
  return (
    <Avatar>
      <AvatarImage src="https://i.pravatar.cc/96?img=12" alt="Team member" />
      <AvatarFallback>FM</AvatarFallback>
    </Avatar>
  );
}
```

## API Reference

### `Avatar`（Root）Props

`Avatar` 继承 `React.ComponentPropsWithoutRef<typeof AvatarPrimitive.Root>`：

| Prop        | 类型      | 默认值 | 说明           |
| ----------- | --------- | ------ | -------------- |
| `className` | `string`  | `-`    | 追加容器样式   |
| `asChild`   | `boolean` | `-`    | 复用子元素渲染 |

### 常用子组件 Props

| 组件             | 关键 Props            | 说明                 |
| ---------------- | --------------------- | -------------------- |
| `AvatarImage`    | `src` `alt`           | 头像图片与替代文本   |
| `AvatarFallback` | `delayMs` `className` | 图片未加载时回退内容 |

## 可访问性

- `AvatarImage` 应始终提供语义化 `alt` 文本。
- `AvatarFallback` 建议使用可识别缩写（如姓名首字母）。
- 在列表场景中建议为外层容器补充用户名称文本，避免仅图像表达身份。
