# Status Badge

`StatusBadge` 是对 `Badge` 的语义化封装，用于以统一视觉表达在线、离线、警告和错误状态。

</div>

```tsx title="components/demos/data-display-demos.tsx"
import { StatusBadge } from "@formaui/components";

export function ServerStatus() {
  return <StatusBadge status="online" />;
}
```

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add status-badge
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { StatusBadge } from "@formaui/components";

export function ServerStatus() {
  return <StatusBadge status="online" />;
}
```

## API Reference

### `StatusBadge` Props

| Prop     | 类型                                            | 默认值 | 说明             |
| -------- | ----------------------------------------------- | ------ | ---------------- |
| `status` | `"online" \| "offline" \| "warning" \| "error"` | `-`    | 状态类型（必填） |

## 可访问性

- 建议与可读文本搭配使用，避免仅靠颜色传达状态。
- 状态值应来自受控业务枚举，避免引入非标准语义。
- 用于关键告警时建议搭配 `Alert` 或 `Result` 提供更完整上下文。
