# Skeleton

`Skeleton` 用于在数据尚未返回时展示占位结构，减少页面跳动并提升感知性能。

组件本身保持极简，只负责脉冲动画与基础外观，具体形状由 `className` 控制。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add skeleton
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { Skeleton } from "@formaui/components";

export function LoadingCards() {
  return (
    <div className="space-y-3 rounded-lg border border-border p-4">
      <Skeleton className="h-5 w-40" />
      <Skeleton className="h-4 w-full" />
      <Skeleton className="h-4 w-2/3" />
    </div>
  );

```

## API Reference

### Props

`Skeleton` 继承 `React.HTMLAttributes<HTMLDivElement>`，常用属性如下：

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `className` | `string` | `-` | 定义占位块尺寸、形状与布局 |
| `children` | `ReactNode` | `-` | 一般不使用，Skeleton 常作为空占位 |

## 可访问性

- `Skeleton` 仅用于加载阶段占位，数据可用后应立即替换为真实内容。
- 建议与 `aria-busy` 或加载文案搭配，避免无语义动画。
- 占位形状应尽量贴近真实布局，减少 CLS（布局偏移）。
