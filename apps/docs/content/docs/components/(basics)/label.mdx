# Label

`Label` 用于为表单控件提供可访问的文本标签，确保用户与辅助技术都能正确理解字段语义。

它基于 `@radix-ui/react-label` 封装，支持 `htmlFor` 关联、必填标识与禁用态样式，适合作为表单的基础文本单元。

## 使用方式

```tsx title="Basic Usage"
import { Label } from "@formaui/components";
import { Input } from "@formaui/components";

export function EmailField() {
  return (
    <div className="grid gap-2">
      <Label htmlFor="team-email" required>
        团队邮箱
      </Label>
      <Input id="team-email" type="email" placeholder="team@company.com" />
    </div>
  );
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `required` | `boolean` | `false` | 在标签后追加可见 `*`，并附带屏幕阅读器可识别的必填提示 |
| `disabled` | `boolean` | `false` | 应用禁用态样式并添加 `aria-disabled` |
| `htmlFor` | `string` | `-` | 关联目标表单控件 `id` |
| `className` | `string` | `-` | 追加自定义样式 |

### 继承属性

`Label` 继承 `React.ComponentPropsWithoutRef<typeof LabelPrimitive.Root>`，可直接使用：

- `onClick`
- `aria-*`
- `data-*`
- `id`

## 可访问性

- 使用 `htmlFor` 与表单控件 `id` 一一对应，确保点击标签可聚焦输入框。
- `required` 同时提供可见标识与 `sr-only` 文本，兼顾视觉与读屏用户。
- 对禁用输入，建议同时设置 `Input disabled` 与 `Label disabled` 保持状态一致。
