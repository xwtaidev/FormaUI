# Image

`Image` 是基于原生 `<img>` 封装的图片组件，提供一致边框样式与 `fallback` 兜底能力。

当主图加载失败时，组件可自动切换到回退地址，避免出现破图占位。

## 使用方式

```tsx title="Basic Usage"
import { Image } from "@formaui/components";

export function CoverImage() {
  return (
    <Image
      src="https://example.com/banner.png"
      fallback="https://example.com/fallback.png"
      alt="Workspace banner"
      className="h-48 w-full object-cover"
    />
  );
}
```

## API Reference

### Props

`Image` 继承 `React.ImgHTMLAttributes<HTMLImageElement>`（不含原生 `onError`，由组件定义为无参函数），常用属性如下：

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `src` | `string \| undefined` | `-` | 主图地址 |
| `alt` | `string \| undefined` | `-` | 替代文本 |
| `fallback` | `string \| undefined` | `-` | 主图失败时切换的兜底图地址 |
| `preview` | `boolean` | `false` | 预览标记（写入 `data-preview`） |
| `onError` | `() => void` | `-` | 触发加载失败时回调 |
| `className` | `string` | `-` | 追加自定义样式 |

## 可访问性

- 必须提供有意义的 `alt` 文本，避免空洞描述。
- 装饰性图片可使用空 `alt=""`，并避免承载关键信息。
- 对失败回退图，建议仍保持准确的 `alt` 语义。
