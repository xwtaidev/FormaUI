# Button

`Button` 是最常用的交互组件，用于触发页面上的主要或次要动作。

它基于语义化 `<button>` 实现，默认继承 FormaUI 的主题变量与焦点样式，适合作为操作入口的标准组件。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add button
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { Button } from "@formaui/components";

export function BasicActions() {
  return (
    <div className="flex items-center gap-2">
      <Button>保存更改</Button>
      <Button variant="outline">取消</Button>
    </div>
  );
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `variant` | `"default" \| "secondary" \| "outline" \| "ghost" \| "destructive"` | `"default"` | 按钮视觉样式 |
| `size` | `"sm" \| "md" \| "lg" \| "icon"` | `"md"` | 按钮尺寸 |
| `className` | `string` | `-` | 追加自定义样式 |
| `disabled` | `boolean` | `false` | 禁用交互并降低可见度 |
| `type` | `"button" \| "submit" \| "reset"` | `"button"`（浏览器默认） | 用于表单提交语义 |

### 继承属性

`Button` 继承 `React.ButtonHTMLAttributes<HTMLButtonElement>`，可直接使用：

- `onClick`
- `aria-*`
- `data-*`
- `name` / `value`

## 可访问性

- 使用原生 `<button>` 元素，默认支持键盘交互。
- 内置 `focus-visible` 样式，满足可见焦点要求。
- `size="icon"` 时必须提供可访问名称（如 `aria-label`）。
