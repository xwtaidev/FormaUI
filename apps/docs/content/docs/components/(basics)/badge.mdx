# Badge

`Badge` 用于展示短文本标识，例如状态、分类、优先级和版本号。

组件提供 `default`、`secondary`、`outline`、`destructive` 四种视觉变体，适合在列表、卡片和工具栏中快速传达信息。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add badge
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { Badge } from "@formaui/components";

export function StatusBadge() {
  return <Badge>Active</Badge>;
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `variant` | `"default" \| "secondary" \| "outline" \| "destructive"` | `"default"` | 标签视觉样式 |
| `className` | `string` | `-` | 追加自定义样式 |

### 继承属性

`Badge` 继承 `React.HTMLAttributes<HTMLDivElement>`，可直接使用：

- `id`
- `onClick`
- `aria-*`
- `data-*`

## 可访问性

- `Badge` 默认用于信息标识，不建议作为唯一交互入口。
- 若作为可点击元素使用，建议改用 `button` 或提供明确键盘交互能力。
- 使用颜色表达状态时，建议搭配文字，避免仅靠颜色区分。
