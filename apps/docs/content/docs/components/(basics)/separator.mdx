# Separator

`Separator` 用于在界面中创建清晰的视觉分段，可用于表单、卡片、工具栏和面板布局。

组件支持横向和纵向方向，并可通过 `decorative` 控制其是否仅作为视觉装饰。

## API Reference

### Props

`Separator` 继承 `React.ComponentPropsWithoutRef<typeof SeparatorPrimitive.Root>`，常用属性如下：

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | 分隔线方向 |
| `decorative` | `boolean` | `true` | 是否仅作视觉装饰 |
| `className` | `string` | `-` | 追加自定义样式 |

## 可访问性

- 纯视觉分隔使用默认 `decorative={true}`。
- 当分隔线表达结构语义时，设置 `decorative={false}`。
- 纵向分隔需确保父容器有明确高度，否则不可见。
