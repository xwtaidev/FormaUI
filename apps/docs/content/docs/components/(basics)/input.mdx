# Input

`Input` 是表单中的基础单行输入组件，适用于邮箱、搜索、密码、编号等短文本场景。

组件基于原生 `<input>` 封装，默认提供一致的尺寸、边框、焦点态与禁用态，可直接用于业务页面。

```bash title="CLI"
npx formaui add input
```


## API Reference

### Props

`Input` 继承 `React.ComponentProps<"input">`，常用属性如下：

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `type` | `string` | `"text"` | 输入类型，例如 `email`、`password`、`search` |
| `placeholder` | `string` | `-` | 占位提示文本 |
| `value` | `string \| number \| readonly string[]` | `-` | 受控值 |
| `defaultValue` | `string \| number \| readonly string[]` | `-` | 非受控初始值 |
| `disabled` | `boolean` | `false` | 禁用交互并降低可见度 |
| `readOnly` | `boolean` | `false` | 只读，不可编辑但可聚焦 |
| `className` | `string` | `-` | 追加自定义样式 |

### 继承属性

可直接使用标准输入属性：

- `onChange`
- `onBlur` / `onFocus`
- `name`
- `aria-*` / `data-*`

## 可访问性

- 建议为输入框提供可见文本标签，并通过 `htmlFor + id` 关联。
- 使用 `type` 表达字段语义（如 `email`、`search`）以提升输入体验。
- 对禁用或只读字段，建议提供辅助说明，避免用户误解为系统异常。
