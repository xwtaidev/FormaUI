# Textarea

`Textarea` 是表单中的多行输入组件，适用于发布备注、问题描述、反馈建议等需要较长文本输入的场景。

组件基于原生 `<textarea>` 封装，默认提供统一边框、焦点环与禁用态表现，并支持通过 `className` 调整高度。



## API Reference

### Props

`Textarea` 继承 `React.ComponentProps<"textarea">`，常用属性如下：

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `placeholder` | `string` | `-` | 占位提示文本 |
| `value` | `string` | `-` | 受控值 |
| `defaultValue` | `string` | `-` | 非受控初始值 |
| `disabled` | `boolean` | `false` | 禁用交互并降低可见度 |
| `readOnly` | `boolean` | `false` | 只读，不可编辑 |
| `rows` | `number` | `-` | 指定可见行数 |
| `className` | `string` | `-` | 追加自定义样式（如调整高度） |

### 继承属性

可直接使用标准多行输入属性：

- `onChange`
- `onBlur` / `onFocus`
- `name`
- `aria-*` / `data-*`

## 可访问性

- 为 `Textarea` 提供明确的文本标签，并通过 `htmlFor + id` 关联。
- 对长度受限的字段，建议增加字数说明或错误提示。
- 对只读/禁用态字段，建议说明“不可编辑原因”，提升可理解性。
