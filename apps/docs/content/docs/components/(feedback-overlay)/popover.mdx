# Popover

`Popover` 用于在触发元素附近展示上下文内容，适合筛选面板、短配置项、轻量操作集合。

## 可视化展示

  <div className="flex min-h-[220px] items-start justify-center overflow-visible p-6 pt-12">
    <PopoverInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/popover-hover-tooltip-demos.tsx"
import {
  Button,
  Popover,
  PopoverContent,
  PopoverTrigger,
} from "@formaui/components";

export function FiltersPopover() {
  return (
    <Popover>
      <PopoverTrigger asChild>
        <Button variant="outline">Open Filters</Button>
      </PopoverTrigger>
      <PopoverContent>Filter controls go here.</PopoverContent>
    </Popover>
  );
}
```

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add popover
```

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import {
  Button,
  Popover,
  PopoverContent,
  PopoverTrigger,
} from "@formaui/components";

export function QuickConfigPopover() {
  return (
    <Popover>
      <PopoverTrigger asChild>
        <Button variant="outline" size="sm">
          快速配置
        </Button>
      </PopoverTrigger>
      <PopoverContent className="w-72">配置项内容</PopoverContent>
    </Popover>
  );
}
```

## API Reference

### `Popover`（Root）Props

`Popover` 继承 `React.ComponentPropsWithoutRef<typeof PopoverPrimitive.Root>`：

| Prop           | 类型                      | 默认值                | 说明                 |
| -------------- | ------------------------- | --------------------- | -------------------- |
| `open`         | `boolean`                 | `-`                   | 受控开关状态         |
| `defaultOpen`  | `boolean`                 | `false`               | 非受控初始状态       |
| `onOpenChange` | `(open: boolean) => void` | `-`                   | 开关变化回调         |
| `modal`        | `boolean`                 | `false`（Radix 默认） | 是否启用模态焦点行为 |

### `PopoverContent` Props

| Prop         | 类型                           | 默认值     | 说明             |
| ------------ | ------------------------------ | ---------- | ---------------- |
| `align`      | `"start" \| "center" \| "end"` | `"center"` | 内容对齐方式     |
| `sideOffset` | `number`                       | `4`        | 与触发器间距     |
| `className`  | `string`                       | `-`        | 内容容器样式扩展 |

## 可访问性

- 具备 Esc 关闭与焦点回退至触发器行为。
- 建议触发器使用有明确语义的按钮文案。
- 浮层内容建议控制在短流程，避免替代完整页面。
