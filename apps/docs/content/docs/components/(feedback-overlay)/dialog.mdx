# Dialog

`Dialog` 用于承载高优先级任务流程，例如删除确认、关键配置编辑、提交前复核。

它基于 Radix Dialog 构建，默认提供遮罩、焦点锁定、Esc 关闭和可访问语义。

```tsx title="components/demos/dialog-drawer-demos.tsx"
import {
  Button,
  Dialog,
  DialogContent,
  DialogDescription,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from "@formaui/components";

export function ConfirmDialog() {
  return (
    <Dialog>
      <DialogTrigger asChild>
        <Button>Open</Button>
      </DialogTrigger>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>Confirm</DialogTitle>
          <DialogDescription>Proceed with this action.</DialogDescription>
        </DialogHeader>
      </DialogContent>
    </Dialog>
  );
}
```

## 使用方式

```tsx title="Basic Usage"
import {
  Button,
  Dialog,
  DialogContent,
  DialogDescription,
  DialogFooter,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from "@formaui/components";

export function PublishConfirmDialog() {
  return (
    <Dialog>
      <DialogTrigger asChild>
        <Button variant="outline">发布变更</Button>
      </DialogTrigger>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>确认发布</DialogTitle>
          <DialogDescription>发布后将同步生产环境。</DialogDescription>
        </DialogHeader>
        <DialogFooter>
          <Button variant="outline">取消</Button>
          <Button>确认发布</Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  );
}
```

## API Reference

### `Dialog`（Root）Props

`Dialog` 继承 `React.ComponentPropsWithoutRef<typeof DialogPrimitive.Root>`：

| Prop           | 类型                      | 默认值               | 说明                 |
| -------------- | ------------------------- | -------------------- | -------------------- |
| `open`         | `boolean`                 | `-`                  | 受控开关状态         |
| `defaultOpen`  | `boolean`                 | `false`              | 非受控初始状态       |
| `onOpenChange` | `(open: boolean) => void` | `-`                  | 开关变化回调         |
| `modal`        | `boolean`                 | `true`（Radix 默认） | 是否启用模态焦点策略 |

### 常用子组件 Props

| 组件                | 关键 Props  | 说明           |
| ------------------- | ----------- | -------------- |
| `DialogTrigger`     | `asChild`   | 触发元素包装   |
| `DialogContent`     | `className` | 对话框内容容器 |
| `DialogTitle`       | `className` | 标题节点       |
| `DialogDescription` | `className` | 描述节点       |
| `DialogClose`       | `asChild`   | 关闭动作包装   |

## 可访问性

- 默认包含焦点锁定与 Esc 关闭行为。
- `DialogTitle` 与 `DialogDescription` 自动参与可访问名称/描述。
- 建议危险操作在标题或描述中明确后果与不可逆性。
- 对话框内主操作与取消操作应有清晰层级。
