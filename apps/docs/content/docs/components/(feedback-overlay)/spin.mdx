# Spin

`Spin` 用于在异步处理中展示加载状态，可包裹内容形成覆盖层，也可单独作为轻量加载指示器。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[320px] items-center justify-center p-6">
    <SpinInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/feedback-structure-demos.tsx"
import { Spin } from "@formaui/components";

export function LoadingPanel() {
  return (
    <Spin spinning tip="Loading report">
      <div className="rounded-md border border-border p-4">Report content</div>
    </Spin>
  );
}
```

## API Reference

### Props

| Prop        | 类型                   | 默认值 | 说明             |
| ----------- | ---------------------- | ------ | ---------------- |
| `spinning`  | `boolean`              | `true` | 是否显示加载状态 |
| `tip`       | `React.ReactNode`      | `-`    | 加载提示文案     |
| `size`      | `"sm" \| "md" \| "lg"` | `"md"` | 加载图标尺寸     |
| `delay`     | `number`               | `0`    | 延迟显示（毫秒） |
| `className` | `string`               | `-`    | 追加容器样式     |

### 继承属性

`Spin` 继承 `React.HTMLAttributes<HTMLDivElement>`，可直接使用：

- `id`
- `aria-*`
- `data-*`
- `onMouseEnter`

## 可访问性

- 根节点根据状态设置 `aria-busy`，便于辅助技术感知加载中状态。
- `tip` 建议使用简短可理解文案（如“加载中…”“同步中…”）。
- 覆盖式加载不应长期阻塞关键交互，超过预期时建议补充错误/重试反馈。
