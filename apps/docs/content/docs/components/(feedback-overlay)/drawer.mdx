# Drawer

`Drawer` 用于从页面边缘滑出内容面板，适合设置面板、详情预览、轻量多步骤任务。

它支持 `top / bottom / left / right` 多方向布局，基于 Radix Dialog 提供焦点与遮罩能力。

## 使用方式

```tsx title="Basic Usage"
import {
  Button,
  Drawer,
  DrawerContent,
  DrawerDescription,
  DrawerFooter,
  DrawerHeader,
  DrawerTitle,
  DrawerTrigger,
} from "@formaui/components";

export function ProfileDrawer() {
  return (
    <Drawer>
      <DrawerTrigger asChild>
        <Button variant="outline">编辑资料</Button>
      </DrawerTrigger>
      <DrawerContent side="right">
        <DrawerHeader>
          <DrawerTitle>Profile Settings</DrawerTitle>
          <DrawerDescription>更新头像、昵称和偏好配置。</DrawerDescription>
        </DrawerHeader>
        <DrawerFooter>
          <Button variant="outline">取消</Button>
          <Button>保存</Button>
        </DrawerFooter>
      </DrawerContent>
    </Drawer>
  );
}
```

## API Reference

### `Drawer`（Root）Props

`Drawer` 继承 `React.ComponentPropsWithoutRef<typeof DialogPrimitive.Root>`：

| Prop           | 类型                      | 默认值               | 说明                 |
| -------------- | ------------------------- | -------------------- | -------------------- |
| `open`         | `boolean`                 | `-`                  | 受控开关状态         |
| `defaultOpen`  | `boolean`                 | `false`              | 非受控初始状态       |
| `onOpenChange` | `(open: boolean) => void` | `-`                  | 开关变化回调         |
| `modal`        | `boolean`                 | `true`（Radix 默认） | 是否启用模态焦点策略 |

### `DrawerContent` Props

| Prop        | 类型                                     | 默认值    | 说明             |
| ----------- | ---------------------------------------- | --------- | ---------------- |
| `side`      | `"top" \| "bottom" \| "left" \| "right"` | `"right"` | 抽屉弹出方向     |
| `className` | `string`                                 | `-`       | 内容容器样式扩展 |

### 常用子组件 Props

| 组件                | 关键 Props  | 说明         |
| ------------------- | ----------- | ------------ |
| `DrawerTrigger`     | `asChild`   | 触发元素包装 |
| `DrawerHeader`      | `className` | 头部容器     |
| `DrawerTitle`       | `className` | 标题节点     |
| `DrawerDescription` | `className` | 描述节点     |
| `DrawerFooter`      | `className` | 底部操作区   |

## 可访问性

- 继承 Dialog 的焦点锁定、Esc 关闭和遮罩语义。
- 标题与描述用于辅助技术理解抽屉上下文。
- 推荐在抽屉内提供清晰主操作与取消操作。
- 移动端底部抽屉建议避免遮挡系统手势区域。
