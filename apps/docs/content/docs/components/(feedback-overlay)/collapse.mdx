# Collapse

`Collapse` 用于在不离开当前页面上下文的情况下展开或收起补充内容。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[260px] items-center justify-center p-6">
    <CollapseInteractiveDemo />

</div>

```tsx title="components/demos/feedback-structure-demos.tsx"
import {
  Collapse,
  CollapseContent,
  CollapseTrigger,
} from "@formaui/components";

export function FiltersCollapse() {
  return (
    <Collapse>
      <CollapseTrigger>Advanced filters</CollapseTrigger>
      <CollapseContent>
        Project status, owner, and date-range selectors.
      </CollapseContent>
    </Collapse>
  );
}
```

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add collapse
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import {
  Collapse,
  CollapseContent,
  CollapseTrigger,
} from "@formaui/components";

export function SettingsCollapse() {
  return (
    <Collapse defaultOpen>
      <CollapseTrigger>高级设置</CollapseTrigger>
      <CollapseContent>通知频率、缓存策略与实验特性配置。</CollapseContent>
    </Collapse>
  );
}
```

## API Reference

### `Collapse`（Root）Props

`Collapse` 继承 `React.ComponentPropsWithoutRef<typeof CollapsiblePrimitive.Root>`：

| Prop           | 类型                      | 默认值  | 说明           |
| -------------- | ------------------------- | ------- | -------------- |
| `open`         | `boolean`                 | `-`     | 受控开关状态   |
| `defaultOpen`  | `boolean`                 | `false` | 非受控初始状态 |
| `onOpenChange` | `(open: boolean) => void` | `-`     | 开关变化回调   |
| `disabled`     | `boolean`                 | `false` | 禁用交互       |

### 常用子组件 Props

| 组件              | 关键 Props  | 说明         |
| ----------------- | ----------- | ------------ |
| `CollapseTrigger` | `className` | 触发器按钮   |
| `CollapseContent` | `className` | 折叠内容容器 |

## 可访问性

- 触发器默认可键盘访问并暴露展开状态。
- 适合承载次级信息，不建议隐藏关键主路径内容。
- 长内容建议分段，避免单次展开信息过密。
