# Alert

`Alert` 用于在页面中就地展示反馈信息，常见于告警提示、操作成功通知、错误反馈等场景。

它支持多种语义变体、图标与附加操作区域，适合与表单、列表、详情页组合使用。

```tsx title="components/demos/alert-demos.tsx"
import { Alert } from "@formaui/components";
import { AlertTriangle } from "lucide-react";

export function BillingAlert() {
  return (
    <Alert
      variant="warning"
      icon={<AlertTriangle />}
      title="Usage threshold reached"
      description="Your project consumed 85% of this month's credits."
    />
  );
}
```

## API Reference

### Props

| Prop          | 类型                                                   | 默认值      | 说明                   |
| ------------- | ------------------------------------------------------ | ----------- | ---------------------- |
| `variant`     | `"default" \| "success" \| "warning" \| "destructive"` | `"default"` | 反馈语义样式           |
| `title`       | `ReactNode`                                            | `-`         | 提示标题               |
| `description` | `ReactNode`                                            | `-`         | 提示描述               |
| `icon`        | `ReactNode`                                            | `-`         | 左侧图标               |
| `children`    | `ReactNode`                                            | `-`         | 扩展内容（如操作按钮） |
| `role`        | `string`                                               | `"status"`  | 辅助技术语义角色       |
| `className`   | `string`                                               | `-`         | 容器样式扩展           |

### 继承属性

`Alert` 继承 `Omit<React.HTMLAttributes<HTMLDivElement>, "title">`，可直接使用：

- `id`
- `aria-*`
- `data-*`
- `onClick`

## 可访问性

- 默认 `role="status"`，可按场景改为 `alert` 等角色。
- 标题与描述分层结构有助于读屏器快速解析信息。
- 搭配按钮动作时，建议按钮文案明确（如“重试”、“查看详情”）。
- `destructive` 场景建议同时给出可恢复路径。
