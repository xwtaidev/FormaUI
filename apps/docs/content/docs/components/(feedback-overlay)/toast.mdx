# Toast

`Toast` 用于展示短时反馈消息，适合保存成功、同步失败、操作撤销等非阻塞提示场景。

```tsx title="components/demos/feedback-structure-demos.tsx"
import {
  Toast,
  ToastClose,
  ToastDescription,
  ToastProvider,
  ToastTitle,
  ToastViewport,
} from "@formaui/components";

export function SaveToast() {
  return (
    <ToastProvider>
      <Toast open>
        <div>
          <ToastTitle>Saved</ToastTitle>
          <ToastDescription>All changes synced.</ToastDescription>
        </div>
        <ToastClose />
      </Toast>
      <ToastViewport />
    </ToastProvider>
  );
}
```

## 使用方式

```tsx title="Basic Usage"
"use client";

import * as React from "react";
import {
  Button,
  Toast,
  ToastClose,
  ToastDescription,
  ToastProvider,
  ToastTitle,
  ToastViewport,
} from "@formaui/components";

export function BasicSaveToast() {
  const [open, setOpen] = React.useState(false);

  return (
    <ToastProvider>
      <Button size="sm" onClick={() => setOpen(true)}>
        保存
      </Button>
      <Toast open={open} onOpenChange={setOpen}>
        <div>
          <ToastTitle>保存成功</ToastTitle>
          <ToastDescription>内容已更新。</ToastDescription>
        </div>
        <ToastClose />
      </Toast>
      <ToastViewport />
    </ToastProvider>
  );
}
```

## API Reference

### `ToastProvider` Props

| Prop       | 类型     | 默认值                         | 说明           |
| ---------- | -------- | ------------------------------ | -------------- |
| `duration` | `number` | `5000`（Radix 默认）           | 默认显示时长   |
| `label`    | `string` | `"Notification"`（Radix 默认） | 无障碍区域标签 |

### `Toast`（Root）Props

| Prop           | 类型                         | 默认值      | 说明           |
| -------------- | ---------------------------- | ----------- | -------------- |
| `open`         | `boolean`                    | `-`         | 受控开关状态   |
| `defaultOpen`  | `boolean`                    | `false`     | 非受控初始状态 |
| `onOpenChange` | `(open: boolean) => void`    | `-`         | 开关变化回调   |
| `variant`      | `"default" \| "destructive"` | `"default"` | 提示样式语义   |

### 常用子组件 Props

| 组件               | 关键 Props  | 说明               |
| ------------------ | ----------- | ------------------ |
| `ToastViewport`    | `className` | 通知容器位置与布局 |
| `ToastTitle`       | `className` | 提示标题           |
| `ToastDescription` | `className` | 提示描述           |
| `ToastClose`       | `className` | 关闭按钮           |
| `ToastAction`      | `altText`   | 可选动作按钮       |

## 可访问性

- 使用专用通知区域语义，支持屏幕阅读器感知。
- 关闭按钮可聚焦并提供读屏标签。
- 提示文案应简洁，避免在 Toast 内承载复杂流程。
