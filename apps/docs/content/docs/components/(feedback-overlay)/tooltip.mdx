# Tooltip

`Tooltip` 用于展示简短提示文案，适合图标按钮含义说明、状态解释和快捷键提示。

## 可视化展示

<div className="not-prose my-4 overflow-visible rounded-xl border border-fd-border">
  <div className="flex min-h-[200px] items-start justify-center overflow-visible p-6 pt-12">
    <TooltipInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/popover-hover-tooltip-demos.tsx"
import {
  Button,
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from "@formaui/components";

export function SaveTooltip() {
  return (
    <TooltipProvider>
      <Tooltip>
        <TooltipTrigger asChild>
          <Button variant="outline" size="sm">
            保存
          </Button>
        </TooltipTrigger>
        <TooltipContent>保存当前草稿</TooltipContent>
      </Tooltip>
    </TooltipProvider>
  );
}
```

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add tooltip
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import {
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from "@formaui/components";

export function BasicTooltip() {
  return (
    <TooltipProvider>
      <Tooltip>
        <TooltipTrigger asChild>
          <span className="underline decoration-dotted">
            什么是 API Credits？
          </span>
        </TooltipTrigger>
        <TooltipContent>用于计量模型调用消耗的额度。</TooltipContent>
      </Tooltip>
    </TooltipProvider>
  );
}
```

## API Reference

### `TooltipProvider` Props

| Prop                | 类型     | 默认值              | 说明                 |
| ------------------- | -------- | ------------------- | -------------------- |
| `delayDuration`     | `number` | `700`（Radix 默认） | 全局打开延迟         |
| `skipDelayDuration` | `number` | `300`（Radix 默认） | 连续触发跳过延迟时长 |

### `Tooltip`（Root）Props

| Prop           | 类型                      | 默认值  | 说明           |
| -------------- | ------------------------- | ------- | -------------- |
| `open`         | `boolean`                 | `-`     | 受控开关状态   |
| `defaultOpen`  | `boolean`                 | `false` | 非受控初始状态 |
| `onOpenChange` | `(open: boolean) => void` | `-`     | 开关变化回调   |

### `TooltipContent` Props

| Prop         | 类型     | 默认值 | 说明           |
| ------------ | -------- | ------ | -------------- |
| `sideOffset` | `number` | `4`    | 与触发器间距   |
| `className`  | `string` | `-`    | 提示层样式扩展 |

## 可访问性

- 建议 Tooltip 文案保持简短，不承载关键操作。
- 触发元素需具备可访问名称（文本或 `aria-label`）。
- 在移动端应避免依赖仅悬停可见的信息。
