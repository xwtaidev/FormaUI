# Hover Card

`HoverCard` 用于在悬停或聚焦时展示补充信息，适合成员预览、标签说明、上下文简介等轻信息场景。

## 可视化展示

<div className="not-prose my-4 overflow-visible rounded-xl border border-fd-border">
  <div className="flex min-h-[220px] items-start justify-center overflow-visible p-6 pt-12">
    <HoverCardInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/popover-hover-tooltip-demos.tsx"
import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from "@formaui/components";

export function MemberPreview() {
  return (
    <HoverCard>
      <HoverCardTrigger>@design-team</HoverCardTrigger>
      <HoverCardContent>Team details and quick links.</HoverCardContent>
    </HoverCard>
  );
}
```

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add hover-card
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from "@formaui/components";

export function TeamHoverCard() {
  return (
    <HoverCard>
      <HoverCardTrigger asChild>
        <a href="#">@frontend-team</a>
      </HoverCardTrigger>
      <HoverCardContent>负责设计系统与组件研发。</HoverCardContent>
    </HoverCard>
  );
}
```

## API Reference

### `HoverCard`（Root）Props

`HoverCard` 继承 `React.ComponentPropsWithoutRef<typeof HoverCardPrimitive.Root>`：

| Prop           | 类型                      | 默认值              | 说明           |
| -------------- | ------------------------- | ------------------- | -------------- |
| `open`         | `boolean`                 | `-`                 | 受控开关状态   |
| `defaultOpen`  | `boolean`                 | `false`             | 非受控初始状态 |
| `onOpenChange` | `(open: boolean) => void` | `-`                 | 开关变化回调   |
| `openDelay`    | `number`                  | `700`（Radix 默认） | 打开延迟       |
| `closeDelay`   | `number`                  | `300`（Radix 默认） | 关闭延迟       |

### `HoverCardContent` Props

| Prop         | 类型                           | 默认值     | 说明             |
| ------------ | ------------------------------ | ---------- | ---------------- |
| `align`      | `"start" \| "center" \| "end"` | `"center"` | 内容对齐方式     |
| `sideOffset` | `number`                       | `4`        | 与触发器间距     |
| `className`  | `string`                       | `-`        | 内容容器样式扩展 |

## 可访问性

- 触发元素支持键盘聚焦，不强制路由跳转。
- 适合补充信息，不建议承载关键确认流程。
- 可通过延迟参数平衡误触与响应速度。
