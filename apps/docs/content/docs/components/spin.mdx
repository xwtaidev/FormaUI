# Spin

## Overview

`Spin` is a loading indicator container for async sections and page fragments.

## Installation

```bash
npx formaui add spin
```

## Usage

```tsx
import { Spin } from "@/components/primitives/spin";

export function LoadingPanel() {
  return (
    <Spin spinning tip="Loading report">
      <div className="rounded-md border border-border p-4">Report content</div>
    </Spin>
  );
}
```

## v0.7 Scope

- Inline and overlay loading state.
- Optional loading text and delay.
- Global route-level loading belongs to Next.js app loading boundaries.
