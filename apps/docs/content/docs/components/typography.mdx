# Typography

## Overview

`Typography` standardizes semantic text styles with a minimal `variant` + `as` API.

## Installation

```bash
npx formaui add typography
```

## Usage

```tsx
import { Typography } from "@/components/primitives/typography";

export function ReleaseSummary() {
  return (
    <div className="space-y-3">
      <Typography variant="h2">v0.6 Release Scope</Typography>
      <Typography variant="lead">Wave A introduces semantic building blocks.</Typography>
      <Typography variant="muted">Use as overrides when integrating into existing design systems.</Typography>
    </div>
  );
}
```

## Accessibility

- Maps text variants to semantic defaults (`h1-h4`, `p`, `small`, `code`).
- Supports explicit `as` overrides without dropping shared visual styling.
- Keeps heading structure predictable for assistive navigation.
