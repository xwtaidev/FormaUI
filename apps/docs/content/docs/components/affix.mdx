# Affix

## Overview

`Affix` keeps an element pinned within viewport boundaries while the page scrolls.

## Installation

```bash
npx formaui add affix
```

## Usage

```tsx
import { Affix } from "@/components/primitives/affix";
import { Button } from "@/components/primitives/button";

export function ToolbarAffix() {
  return (
    <Affix offsetTop={12}>
      <Button size="sm">Quick Action</Button>
    </Affix>
  );
}
```

## v0.7 Scope

- Sticky positioning with top/bottom offsets.
- Optional affixed state callback.
- Complex container nesting and portal offsets are out of scope for v0.7.
