# Date Picker

`DatePicker` 适用于截止日期、计划日期、发布日期等单日期选择场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add date-picker
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { DatePicker } from "@formaui/components";

export function ReleaseDateField() {
  return <DatePicker placeholder="选择发布日期" />;
}
```

## API Reference

### Props

| Prop           | 类型                                 | 默认值          | 说明           |
| -------------- | ------------------------------------ | --------------- | -------------- |
| `value`        | `Date`                               | `-`             | 受控选中日期   |
| `defaultValue` | `Date`                               | `-`             | 非受控初始日期 |
| `onChange`     | `(value: Date \| undefined) => void` | `-`             | 日期变更回调   |
| `placeholder`  | `string`                             | `"Pick a date"` | 占位文案       |
| `disabled`     | `boolean`                            | `false`         | 禁用触发器     |
| `className`    | `string`                             | `-`             | 触发器样式扩展 |

## 可访问性

- 触发器使用按钮语义，支持键盘打开日历。
- 已选日期支持清空操作，避免强制保留错误值。
- 建议在表单中搭配字段标签说明日期含义。
