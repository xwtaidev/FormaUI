# Slider

`Slider` 适合音量、阈值、价格区间等连续值输入场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add slider
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
"use client";

import * as React from "react";
import { Slider } from "@formaui/components";

export function VolumeField() {
  const [value, setValue] = React.useState([50]);
  return <Slider value={value} onValueChange={setValue} max={100} step={1} />;
}
```

## API Reference

### Props

`Slider` 继承 `React.ComponentPropsWithoutRef<typeof SliderPrimitive.Root>`，常用参数如下：

| Prop            | 类型                        | 默认值  | 说明                    |
| --------------- | --------------------------- | ------- | ----------------------- |
| `value`         | `number[]`                  | `-`     | 受控值（支持单值/区间） |
| `defaultValue`  | `number[]`                  | `-`     | 非受控初始值            |
| `onValueChange` | `(value: number[]) => void` | `-`     | 值变化回调              |
| `min`           | `number`                    | `0`     | 最小值                  |
| `max`           | `number`                    | `100`   | 最大值                  |
| `step`          | `number`                    | `1`     | 步进粒度                |
| `disabled`      | `boolean`                   | `false` | 禁用交互                |
| `className`     | `string`                    | `-`     | 根容器样式扩展          |

## 可访问性

- 基于 Radix Slider，支持键盘方向键调节。
- 区间滑块建议搭配数值文本同步显示。
- 对关键参数建议增加单位说明（如 `%`、`ms`）。
