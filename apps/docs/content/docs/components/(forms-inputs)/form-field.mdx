# Form Field

`FormField` 用于包装输入控件并统一字段标签、描述文案与错误提示的显示逻辑。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add form-field
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { FormField, Input } from "@formaui/components";

export function DisplayNameField() {
  return (
    <FormField
      label="显示名称"
      htmlFor="display-name"
      description="将用于团队协作展示"
    >
      <Input id="display-name" placeholder="请输入显示名称" />
    </FormField>
  );
}
```

## API Reference

### Props

| Prop          | 类型        | 默认值 | 说明                           |
| ------------- | ----------- | ------ | ------------------------------ |
| `label`       | `string`    | `-`    | 字段标签（必填）               |
| `htmlFor`     | `string`    | `-`    | 关联输入控件 ID（必填）        |
| `description` | `string`    | `-`    | 辅助说明文案                   |
| `error`       | `string`    | `-`    | 错误文案（优先于 description） |
| `children`    | `ReactNode` | `-`    | 字段控件内容                   |
| `className`   | `string`    | `-`    | 外层容器样式                   |

## 可访问性

- 标签通过 `htmlFor` 与控件 `id` 关联，确保语义正确。
- 错误文案与描述文案互斥，避免信息冲突。
- 建议错误文案精确描述修复动作。
