# Color Picker

`ColorPicker` 用于选择主题色、品牌色等视觉配置，支持色板选择与 HEX 文本输入。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add color-picker
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { ColorPicker } from "@formaui/components";

export function ThemeColorField() {
  return <ColorPicker defaultValue="#1677ff" />;
}
```

## API Reference

### Props

| Prop            | 类型                      | 默认值      | 说明         |
| --------------- | ------------------------- | ----------- | ------------ |
| `value`         | `string`                  | `-`         | 受控颜色值   |
| `defaultValue`  | `string`                  | `"#1677ff"` | 非受控初始值 |
| `onValueChange` | `(value: string) => void` | `-`         | 颜色变更回调 |
| `disabled`      | `boolean`                 | `false`     | 禁用交互     |
| `className`     | `string`                  | `-`         | 外层容器样式 |

## 可访问性

- 色板输入使用 `input[type="color"]`，支持原生辅助能力。
- 文本输入建议保持 HEX 格式，避免非预期颜色值。
- 禁用状态会同时作用于色板与文本输入。
