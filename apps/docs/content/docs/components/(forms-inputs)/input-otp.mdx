# Input OTP

`InputOtp` 适用于登录验证、敏感操作确认等验证码输入场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add input-otp
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { InputOtp } from "@formaui/components";

export function VerifyCodeField() {
  return <InputOtp length={6} />;
}
```

## API Reference

### Props

| Prop             | 类型                      | 默认值  | 说明             |
| ---------------- | ------------------------- | ------- | ---------------- |
| `value`          | `string`                  | `-`     | 受控验证码值     |
| `defaultValue`   | `string`                  | `""`    | 非受控初始值     |
| `length`         | `number`                  | `6`     | 验证码位数       |
| `onChange`       | `(value: string) => void` | `-`     | 值变更回调       |
| `disabled`       | `boolean`                 | `false` | 禁用输入         |
| `inputClassName` | `string`                  | `-`     | 单格输入样式扩展 |
| `className`      | `string`                  | `-`     | 容器样式扩展     |

## 可访问性

- 每个输入格带有 `aria-label`（Digit N）。
- 支持左右方向键切换焦点与 Backspace 回退。
- 支持粘贴自动分发，提高键盘与屏幕阅读器用户效率。
