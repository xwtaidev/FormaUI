# Upload

`Upload` 适用于头像、附件、报告等单文件上传场景，内置基础本地校验能力。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add upload
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { Upload } from "@formaui/components";

export function AvatarUploadField() {
  return (
    <Upload
      allowedTypes={["image/png", "image/jpeg"]}
      maxSizeInBytes={2 * 1024 * 1024}
      helperText="仅支持 PNG/JPEG，最大 2MB"
    />
  );
}
```

## API Reference

### Props

| Prop                | 类型                                | 默认值                                           | 说明               |
| ------------------- | ----------------------------------- | ------------------------------------------------ | ------------------ |
| `value`             | `File \| null`                      | `-`                                              | 受控文件值         |
| `defaultValue`      | `File \| null`                      | `null`                                           | 非受控初始值       |
| `onValueChange`     | `(file: File \| null) => void`      | `-`                                              | 文件变更回调       |
| `onValidationError` | `(message: string \| null) => void` | `-`                                              | 校验错误回调       |
| `maxSizeInBytes`    | `number`                            | `-`                                              | 文件大小上限       |
| `allowedTypes`      | `string[]`                          | `-`                                              | 允许的 MIME 类型   |
| `helperText`        | `string`                            | `"Single file upload. Validation runs locally."` | 辅助文案           |
| `clearLabel`        | `string`                            | `"Clear file"`                                   | 清除按钮无障碍文案 |
| `disabled`          | `boolean`                           | `false`                                          | 禁用交互           |
| `className`         | `string`                            | `-`                                              | 容器样式扩展       |
| `inputClassName`    | `string`                            | `-`                                              | 输入框样式扩展     |

## 可访问性

- 文件输入提供默认 `aria-label`，可按场景覆盖。
- 校验错误通过 `role="alert"` 输出，便于读屏器感知。
- 已选文件展示支持单独移除按钮。
