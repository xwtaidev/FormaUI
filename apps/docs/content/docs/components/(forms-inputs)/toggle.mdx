# Toggle

`Toggle` 用于单个命令按钮的开关状态切换，例如加粗、收藏、启用项。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add toggle
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { Toggle } from "@formaui/components";

export function BoldToggle() {
  return <Toggle aria-label="Toggle bold">B</Toggle>;
}
```

## API Reference

### Props

`Toggle` 继承 `React.ComponentPropsWithoutRef<typeof TogglePrimitive.Root>`：

| Prop              | 类型                         | 默认值      | 说明           |
| ----------------- | ---------------------------- | ----------- | -------------- |
| `pressed`         | `boolean`                    | `-`         | 受控开关状态   |
| `defaultPressed`  | `boolean`                    | `false`     | 非受控初始状态 |
| `onPressedChange` | `(pressed: boolean) => void` | `-`         | 状态变化回调   |
| `variant`         | `"default" \| "outline"`     | `"default"` | 视觉样式       |
| `size`            | `"sm" \| "md" \| "lg"`       | `"md"`      | 尺寸           |
| `disabled`        | `boolean`                    | `false`     | 禁用交互       |

## 可访问性

- 状态通过 `aria-pressed` 自动暴露给辅助技术。
- 建议始终提供可读标签或 `aria-label`。
- 用于关键行为时建议增加文案反馈。
