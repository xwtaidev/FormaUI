# Toggle

## Overview

`Toggle` is a two-state pressed control for inline on/off actions.

## Installation

```bash
npx formaui add toggle
```

## Usage

```tsx
import { Toggle } from "@/components/primitives/toggle";

export function PinToggle() {
  return <Toggle aria-label="Pin filters">Pin</Toggle>;
}
```

## Accessibility

- Maintains pressed state with Radix toggle semantics.
- Keyboard focus and activation are supported by default.
- Works for icon-only or text toggles with `aria-label`.
