# Toggle Group

`ToggleGroup` 适用于文本样式、对齐方式、过滤标签等集合型切换场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add toggle-group
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
"use client";

import * as React from "react";
import { ToggleGroup, ToggleGroupItem } from "@formaui/components";

export function AlignField() {
  const [value, setValue] = React.useState("center");
  return (
    <ToggleGroup
      type="single"
      value={value}
      onValueChange={(next) => next && setValue(next)}
    >
      <ToggleGroupItem value="left" aria-label="Left">
        左
      </ToggleGroupItem>
      <ToggleGroupItem value="center" aria-label="Center">
        中
      </ToggleGroupItem>
      <ToggleGroupItem value="right" aria-label="Right">
        右
      </ToggleGroupItem>
    </ToggleGroup>
  );
}
```

## API Reference

### `ToggleGroup` Props

`ToggleGroup` 继承 `React.ComponentPropsWithoutRef<typeof ToggleGroupPrimitive.Root>`，核心属性：

| Prop            | 类型                     | 说明             |
| --------------- | ------------------------ | ---------------- |
| `type`          | `"single" \| "multiple"` | 选择模式（必填） |
| `value`         | `string \| string[]`     | 受控值           |
| `defaultValue`  | `string \| string[]`     | 非受控初始值     |
| `onValueChange` | `(value: any) => void`   | 值变化回调       |
| `disabled`      | `boolean`                | 禁用整个组       |

### `ToggleGroupItem` Props

| Prop       | 类型                     | 默认值      | 说明           |
| ---------- | ------------------------ | ----------- | -------------- |
| `value`    | `string`                 | `-`         | 选项值（必填） |
| `variant`  | `"default" \| "outline"` | `"default"` | 视觉样式       |
| `size`     | `"sm" \| "md" \| "lg"`   | `"md"`      | 尺寸           |
| `disabled` | `boolean`                | `false`     | 禁用当前项     |

## 可访问性

- 基于 Radix ToggleGroup，支持键盘导航。
- 单选模式建议用互斥含义明确的文案。
- 为每项提供 `aria-label` 便于读屏器朗读。
