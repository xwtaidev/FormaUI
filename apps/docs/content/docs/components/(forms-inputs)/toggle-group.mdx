# Toggle Group

## Overview

`ToggleGroup` coordinates multiple toggles in single or multiple selection mode.

## Installation

```bash
npx formaui add toggle-group
```

## Usage

```tsx
import { ToggleGroup, ToggleGroupItem } from "@/components/primitives/toggle-group";

export function StatusFilter() {
  return (
    <ToggleGroup type="single" defaultValue="all" aria-label="Filter mode">
      <ToggleGroupItem value="all" aria-label="All items">
        All
      </ToggleGroupItem>
      <ToggleGroupItem value="open" aria-label="Open items">
        Open
      </ToggleGroupItem>
    </ToggleGroup>
  );
}
```

## Accessibility

- Radix roving focus enables keyboard traversal.
- `type="single"` maps to radio-like selection behavior.
- Shares variants with `toggle` for consistent states.
