# Combobox

`Combobox` 结合触发按钮、搜索输入与候选列表，适合负责人选择、成员检索、标签选择等查找场景。

它支持受控与非受控模式，内置关键词过滤、空态文案和禁用选项处理。

```tsx title="components/demos/select-suite-demos.tsx"
const ownerOptions = [
  { value: "alpha", label: "Alpha Team" },
  { value: "beta", label: "Beta Team" },
];

export function OwnerPicker() {
  return (
    <Combobox
      options={ownerOptions}
      placeholder="Select owner"
      searchPlaceholder="Search owners"
    />
  );
}
```

## API Reference

### Props

| Prop                | 类型                      | 默认值                | 说明               |
| ------------------- | ------------------------- | --------------------- | ------------------ |
| `options`           | `ComboboxOption[]`        | `-`                   | 候选项列表（必填） |
| `value`             | `string`                  | `-`                   | 受控选中值         |
| `defaultValue`      | `string`                  | `""`                  | 非受控初始值       |
| `onValueChange`     | `(value: string) => void` | `-`                   | 选中值变更回调     |
| `placeholder`       | `string`                  | `"Select an option"`  | 按钮占位文案       |
| `searchPlaceholder` | `string`                  | `"Search options"`    | 搜索输入占位文案   |
| `emptyMessage`      | `string`                  | `"No options found."` | 搜索无结果文案     |
| `disabled`          | `boolean`                 | `false`               | 禁用交互           |
| `className`         | `string`                  | `-`                   | 外层容器样式       |

### `ComboboxOption` 结构

| 字段       | 类型       | 说明           |
| ---------- | ---------- | -------------- |
| `value`    | `string`   | 选项值         |
| `label`    | `string`   | 选项文案       |
| `keywords` | `string[]` | 额外搜索关键词 |
| `disabled` | `boolean`  | 禁用当前选项   |

## 可访问性

- 触发器使用 `role="combobox"` 并暴露展开状态。
- 候选列表使用 `role="listbox"`，选项使用 `role="option"`。
- 搜索输入通过 `aria-label` 提供明确语义。
- 空态文案应描述“无结果”而非留空，避免用户困惑。
