# Input Number

`InputNumber` 适用于配额、预算、数量等数值输入场景，支持按钮步进与键盘方向键操作。

组件内置 `min/max` 边界钳制与 `step` 增减逻辑，支持受控与非受控两种模式。


## API Reference

### Props

| Prop            | 类型                      | 默认值       | 说明               |
| --------------- | ------------------------- | ------------ | ------------------ |
| `value`         | `number`                  | `-`          | 受控数值           |
| `defaultValue`  | `number`                  | `min` 或 `0` | 非受控初始值       |
| `min`           | `number`                  | `-`          | 最小边界           |
| `max`           | `number`                  | `-`          | 最大边界           |
| `step`          | `number`                  | `1`          | 每次增减步长       |
| `onValueChange` | `(value: number) => void` | `-`          | 数值变更回调       |
| `disabled`      | `boolean`                 | `false`      | 禁用输入与增减按钮 |
| `className`     | `string`                  | `-`          | 输入框样式扩展     |

### 继承属性

`InputNumber` 继承 `Omit<React.InputHTMLAttributes<HTMLInputElement>, ...>`，可直接使用常见原生属性：

- `name`
- `id`
- `aria-*`
- `onFocus` / `onBlur`

## 可访问性

- 支持 `ArrowUp` / `ArrowDown` 键盘步进。
- 保留原生 `input[type="number"]` 语义，便于辅助技术识别。
- 内置增减按钮分别带有 `aria-label`，提升可读性。
- 对关键字段建议同时展示单位或范围说明。
