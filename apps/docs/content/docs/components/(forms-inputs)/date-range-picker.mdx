# Date Range Picker

`DateRangePicker` 用于选择开始与结束日期，适合报表筛选、活动周期设置等场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add date-range-picker
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { DateRangePicker } from "@formaui/components";

export function ReportRangeField() {
  return (
    <DateRangePicker defaultValue={{ from: "2026-05-01", to: "2026-05-31" }} />
  );
}
```

## API Reference

### Props

| Prop           | 类型                              | 默认值         | 说明           |
| -------------- | --------------------------------- | -------------- | -------------- |
| `value`        | `Partial<DateRangeValue>`         | `-`            | 受控区间       |
| `defaultValue` | `Partial<DateRangeValue>`         | `-`            | 非受控初始区间 |
| `onChange`     | `(value: DateRangeValue) => void` | `-`            | 区间变更回调   |
| `fromLabel`    | `string`                          | `"Start date"` | 起始日期标签   |
| `toLabel`      | `string`                          | `"End date"`   | 结束日期标签   |
| `className`    | `string`                          | `-`            | 容器样式扩展   |

### `DateRangeValue`

| 字段   | 类型     | 说明                   |
| ------ | -------- | ---------------------- |
| `from` | `string` | 起始日期（YYYY-MM-DD） |
| `to`   | `string` | 结束日期（YYYY-MM-DD） |

## 可访问性

- 起止日期均有显式标签，便于辅助技术识别。
- 结束日期输入会自动带 `min=from`，减少非法范围。
- 建议在业务层补充“结束时间不早于开始时间”的校验提示。
