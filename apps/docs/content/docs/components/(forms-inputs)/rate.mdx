# Rate

`Rate` 适用于满意度评价、内容评分等需要星级反馈的场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add rate
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { Rate } from "@formaui/components";

export function ReviewScoreField() {
  return <Rate allowHalf defaultValue={4.5} />;
}
```

## API Reference

### Props

| Prop            | 类型                      | 默认值  | 说明           |
| --------------- | ------------------------- | ------- | -------------- |
| `value`         | `number`                  | `-`     | 受控评分值     |
| `defaultValue`  | `number`                  | `0`     | 非受控初始评分 |
| `onValueChange` | `(value: number) => void` | `-`     | 评分变化回调   |
| `count`         | `number`                  | `5`     | 星星总数       |
| `allowHalf`     | `boolean`                 | `false` | 是否启用半星   |
| `disabled`      | `boolean`                 | `false` | 禁用交互       |
| `className`     | `string`                  | `-`     | 容器样式扩展   |

## 可访问性

- 每个评分按钮使用 `role="radio"`，整体容器为 `role="radiogroup"`。
- 每个评分项带有 `aria-label`，便于语音输出。
- 半星模式建议搭配文本提示，减少视觉歧义。
