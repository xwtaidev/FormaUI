# Segmented

`Segmented` 适用于视图模式、密度、时间维度等少量互斥选项切换场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add segmented
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { Segmented } from "@formaui/components";

export function ViewModeField() {
  return (
    <Segmented
      defaultValue="list"
      options={[
        { value: "list", label: "列表" },
        { value: "board", label: "看板" },
      ]}
    />
  );
}
```

## API Reference

### Props

| Prop            | 类型                      | 默认值  | 说明             |
| --------------- | ------------------------- | ------- | ---------------- |
| `options`       | `SegmentedOption[]`       | `-`     | 选项列表（必填） |
| `value`         | `string`                  | `-`     | 受控选中值       |
| `defaultValue`  | `string`                  | `-`     | 非受控初始值     |
| `onValueChange` | `(value: string) => void` | `-`     | 值变化回调       |
| `disabled`      | `boolean`                 | `false` | 禁用整个控件     |
| `size`          | `"sm" \| "md"`            | `"md"`  | 选项尺寸         |
| `className`     | `string`                  | `-`     | 容器样式扩展     |

### `SegmentedOption`

| 字段       | 类型        | 说明         |
| ---------- | ----------- | ------------ |
| `label`    | `ReactNode` | 展示文案     |
| `value`    | `string`    | 选项值       |
| `disabled` | `boolean`   | 禁用当前选项 |

## 可访问性

- 基于 Radix ToggleGroup 单选模式实现，支持键盘切换。
- 建议保持 2~5 个选项，确保可快速扫描。
- 每个选项应使用简短且互斥的文案。
