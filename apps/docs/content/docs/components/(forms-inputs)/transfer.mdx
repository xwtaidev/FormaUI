# Transfer

`Transfer` 适用于权限分配、成员分组、资源授权等双列表迁移场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add transfer
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
"use client";

import * as React from "react";
import { Transfer } from "@formaui/components";

const data = [
  { key: "read", title: "读取数据" },
  { key: "write", title: "编辑内容" },
];

export function PermissionTransfer() {
  const [targetKeys, setTargetKeys] = React.useState<string[]>(["read"]);
  return (
    <Transfer data={data} targetKeys={targetKeys} onChange={setTargetKeys} />
  );
}
```

## API Reference

### Props

| Prop             | 类型                                           | 默认值  | 说明                      |
| ---------------- | ---------------------------------------------- | ------- | ------------------------- |
| `data`           | `TransferItem[]`                               | `-`     | 全量数据列表（必填）      |
| `targetKeys`     | `string[]`                                     | `-`     | 右侧已选 key 列表（必填） |
| `selectedKeys`   | `string[]`                                     | `-`     | 受控选中项 key 列表       |
| `onChange`       | `(targetKeys: string[]) => void`               | `-`     | 右侧列表变化回调（必填）  |
| `onSelectChange` | `(source: string[], target: string[]) => void` | `-`     | 选中项变化回调            |
| `disabled`       | `boolean`                                      | `false` | 禁用整体交互              |
| `className`      | `string`                                       | `-`     | 容器样式扩展              |

### `TransferItem`

| 字段       | 类型      | 说明       |
| ---------- | --------- | ---------- |
| `key`      | `string`  | 唯一标识   |
| `title`    | `string`  | 展示文案   |
| `disabled` | `boolean` | 禁用当前项 |

## 可访问性

- 每项通过原生 checkbox 表达选中状态。
- 操作按钮提供明确 `aria-label`（左右移动）。
- 建议对“无项目”状态提供解释文案。
