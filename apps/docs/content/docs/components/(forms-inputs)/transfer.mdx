# Transfer

## Overview

`Transfer` moves records between source and target lists for dual-list assignment workflows.

## Installation

```bash
npx formaui add transfer
```

## Usage

```tsx
import { useState } from "react";
import { Transfer } from "@/components/primitives/transfer";

const items = [
  { key: "1", title: "Alpha Team" },
  { key: "2", title: "Beta Team" }
];

export function TeamTransfer() {
  const [targetKeys, setTargetKeys] = useState<string[]>(["2"]);
  return <Transfer data={items} targetKeys={targetKeys} onChange={setTargetKeys} />;
}
```

## v0.7 Scope

- Manual move left/right actions with selected state.
- Supports controlled `targetKeys`.
- Search/filter and pagination are out of scope for v0.7.
