# RadioGroup

`RadioGroup` 与 `RadioGroupItem` 用于处理「互斥单选」场景，常见于状态筛选、模式切换、方案选择等交互。

它基于 Radix Radio Group 实现，内置键盘导航与焦点样式，并继承 FormaUI 主题变量。

## API Reference

### `RadioGroup` Props

| Prop            | 类型                      | 默认值  | 说明           |
| --------------- | ------------------------- | ------- | -------------- |
| `value`         | `string`                  | `-`     | 受控选中值     |
| `defaultValue`  | `string`                  | `-`     | 非受控初始值   |
| `onValueChange` | `(value: string) => void` | `-`     | 选中项变更回调 |
| `name`          | `string`                  | `-`     | 表单字段名     |
| `required`      | `boolean`                 | `false` | 表单必填校验   |
| `disabled`      | `boolean`                 | `false` | 禁用整个分组   |
| `className`     | `string`                  | `-`     | 追加自定义样式 |

### `RadioGroupItem` Props

| Prop        | 类型      | 默认值  | 说明             |
| ----------- | --------- | ------- | ---------------- |
| `value`     | `string`  | `-`     | 单选项值（必填） |
| `id`        | `string`  | `-`     | 与 label 关联    |
| `disabled`  | `boolean` | `false` | 禁用当前选项     |
| `className` | `string`  | `-`     | 追加自定义样式   |

### 继承属性

- `RadioGroup` 继承 `React.ComponentPropsWithoutRef<typeof RadioGroupPrimitive.Root>`
- `RadioGroupItem` 继承 `React.ComponentPropsWithoutRef<typeof RadioGroupPrimitive.Item>`

常用扩展属性：

- `aria-*`
- `data-*`
- `onFocus` / `onBlur`

## 可访问性

- 基于 Radix Radio Group，支持键盘方向键切换与焦点管理。
- 每个 `RadioGroupItem` 必须有清晰文案标签，并通过 `id` 与 `label` 关联。
- 同一分组内选项应保持语义互斥，避免出现含义重叠选项。
- 默认选项建议显式设置，避免“未选择”导致提交歧义。
