# Checkbox

`Checkbox` 用于处理「可多选」场景，适合设置项、权限矩阵、筛选条件等交互。

它基于 Radix Checkbox 构建，支持 `checked`、`unchecked` 与 `indeterminate` 三种状态，并继承 FormaUI 的主题与焦点样式。

```tsx title="Permissions with Indeterminate"
"use client";

import * as React from "react";
import { Checkbox } from "@formaui/components";

const items = ["read", "write", "publish"] as const;

export function PermissionsSelector() {
  const [selected, setSelected] = React.useState<string[]>(["read"]);
  const allChecked = selected.length === items.length;
  const indeterminate = selected.length > 0 && !allChecked;

  return (
    <label htmlFor="all" className="flex items-start gap-3">
      <Checkbox
        id="all"
        checked={allChecked ? true : indeterminate ? "indeterminate" : false}
        onCheckedChange={(checked) => {
          setSelected(checked === true ? [...items] : []);
        }}
      />
      <span className="text-sm">全选权限</span>
    </label>
  );
}
```

## API Reference

### Props

| Prop              | 类型                                            | 默认值  | 说明                 |
| ----------------- | ----------------------------------------------- | ------- | -------------------- |
| `checked`         | `boolean \| "indeterminate"`                    | `-`     | 受控状态值，支持半选 |
| `defaultChecked`  | `boolean`                                       | `false` | 非受控初始状态       |
| `onCheckedChange` | `(checked: boolean \| "indeterminate") => void` | `-`     | 状态变更回调         |
| `disabled`        | `boolean`                                       | `false` | 禁用交互并降低可见度 |
| `required`        | `boolean`                                       | `false` | 表单必填校验         |
| `name`            | `string`                                        | `-`     | 表单字段名           |
| `value`           | `string`                                        | `"on"`  | 表单提交值           |
| `id`              | `string`                                        | `-`     | 用于关联可点击标签   |
| `className`       | `string`                                        | `-`     | 追加自定义样式       |

### 继承属性

`Checkbox` 继承 `React.ComponentPropsWithoutRef<typeof CheckboxPrimitive.Root>`，可直接使用：

- `aria-*`
- `data-*`
- `onFocus` / `onBlur`
- `onKeyDown`

## 可访问性

- 基于 Radix Checkbox，默认提供 `role="checkbox"` 与键盘交互支持。
- 为每个 `Checkbox` 提供清晰的文本标签，避免无语义图标作为唯一说明。
- 父级「全选」状态建议使用 `indeterminate` 反馈子项部分选中。
- 禁用状态仅用于不可操作场景，避免误导用户。
