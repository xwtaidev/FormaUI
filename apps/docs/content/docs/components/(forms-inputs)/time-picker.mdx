# Time Picker

`TimePicker` 适用于会议时间、提醒时间、营业时段等时刻输入场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add time-picker
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { TimePicker } from "@formaui/components";

export function MeetingTimeField() {
  return <TimePicker defaultValue="09:30" step={300} />;
}
```

## API Reference

### Props

| Prop            | 类型                      | 默认值  | 说明                  |
| --------------- | ------------------------- | ------- | --------------------- |
| `value`         | `string`                  | `-`     | 受控时间值（`HH:mm`） |
| `defaultValue`  | `string`                  | `""`    | 非受控初始值          |
| `onValueChange` | `(value: string) => void` | `-`     | 时间变更回调          |
| `format`        | `"24h" \| "12h"`          | `"24h"` | 显示格式              |
| `step`          | `number`                  | `60`    | 秒级步进              |
| `disabled`      | `boolean`                 | `false` | 禁用交互              |
| `className`     | `string`                  | `-`     | 输入样式扩展          |

## 可访问性

- 使用 `input[type="time"]` 保持原生语义。
- 左侧图标仅作视觉辅助，不替代字段标签。
- 建议在表单外层补充时区说明。
