# Combobox

## Overview

`Combobox` combines a trigger button, searchable input, and selectable option list for compact lookup flows.

## Installation

```bash
npx formaui add combobox
```

## Usage

```tsx
import { Combobox } from "@/components/primitives/combobox";

const ownerOptions = [
  { value: "alpha", label: "Alpha Team" },
  { value: "beta", label: "Beta Team" }
];

export function OwnerPicker() {
  return <Combobox options={ownerOptions} placeholder="Select owner" onValueChange={console.log} />;
}
```

## Accessibility

- Trigger uses `role="combobox"` with expanded state.
- Option list is exposed as `role="listbox"` and each item as `role="option"`.
- Search input has explicit label text for screen readers.
