# Pagination Bar

`PaginationBar` 提供页码切换、页大小切换与总量展示，适合中后台数据列表场景。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[220px] items-center justify-center p-6">
    <PaginationBarInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/workflows-demos.tsx"
import { PaginationBar } from "@formaui/components";

export function TeamPagination() {
  return (
    <PaginationBar
      totalItems={128}
      defaultPage={1}
      defaultPageSize={20}
      onPageChange={(page) => console.log(page)}
      onPageSizeChange={(size) => console.log(size)}
    />
  );
}
```

## API Reference

### `PaginationBar` Props

| Prop               | 类型                         | 默认值         | 说明               |
| ------------------ | ---------------------------- | -------------- | ------------------ |
| `totalItems`       | `number`                     | `-`            | 总数据量（必填）   |
| `page`             | `number`                     | `-`            | 受控当前页         |
| `defaultPage`      | `number`                     | `1`            | 非受控初始页       |
| `pageSize`         | `number`                     | `-`            | 受控每页数量       |
| `defaultPageSize`  | `number`                     | `10`           | 非受控初始每页数量 |
| `pageSizeOptions`  | `number[]`                   | `[10, 20, 50]` | 每页数量候选项     |
| `onPageChange`     | `(page: number) => void`     | `-`            | 页码变化回调       |
| `onPageSizeChange` | `(pageSize: number) => void` | `-`            | 每页数量变化回调   |
| `className`        | `string`                     | `-`            | 追加容器样式       |

## 可访问性

- 组件根节点为 `nav` 且包含 `aria-label`，便于辅助技术识别。
- 页码文案区域使用 `aria-live="polite"` 同步分页状态。
- 上一页/下一页按钮在边界条件下自动禁用，降低误操作。
