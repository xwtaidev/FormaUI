# Data Table

`DataTable` 用于构建可排序数据表，支持列对齐、自定义渲染和空数据回退。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[320px] items-center justify-center p-6">
    <DataTableInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/workflows-demos.tsx"
import { DataTable } from "@formaui/components";

const rows = [
  { name: "Avery Lin", role: "Product Lead", score: 98 },
  { name: "Riley Chen", role: "ML Engineer", score: 92 },
];

export function TeamTable() {
  return (
    <DataTable
      columns={[
        { key: "name", header: "Name", sortable: true },
        { key: "role", header: "Role" },
        { key: "score", header: "Score", sortable: true, align: "right" },
      ]}
      rows={rows}
    />
  );
}
```

## API Reference

### `DataTable` Props

| Prop           | 类型                       | 默认值                | 说明           |
| -------------- | -------------------------- | --------------------- | -------------- |
| `columns`      | `DataTableColumn<TData>[]` | `-`                   | 列配置（必填） |
| `rows`         | `TData[]`                  | `-`                   | 行数据（必填） |
| `caption`      | `string`                   | `-`                   | 表格标题       |
| `className`    | `string`                   | `-`                   | 追加容器样式   |
| `emptyMessage` | `string`                   | `"No records found."` | 空数据提示文案 |

### `DataTableColumn` 结构

| 字段        | 类型                                  | 默认值   | 说明               |
| ----------- | ------------------------------------- | -------- | ------------------ |
| `key`       | `Extract<keyof TData, string>`        | `-`      | 列字段键（必填）   |
| `header`    | `string`                              | `-`      | 列头文案（必填）   |
| `sortable`  | `boolean`                             | `false`  | 是否可排序         |
| `align`     | `"left" \| "center" \| "right"`       | `"left"` | 列内容对齐方式     |
| `className` | `string`                              | `-`      | 追加列样式         |
| `render`    | `(value, row, rowIndex) => ReactNode` | `-`      | 自定义单元格渲染器 |

## 可访问性

- 基于原生 `<table>` 语义，适合表格类读屏场景。
- 可排序列使用按钮触发，支持键盘激活。
- 建议为关键数据列保持稳定顺序，避免频繁重排影响可读性。
