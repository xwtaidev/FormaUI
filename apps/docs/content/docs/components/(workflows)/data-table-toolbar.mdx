# Data Table Toolbar

`DataTableToolbar` 提供表格场景常用的查询、结果统计与操作按钮组合。

    <DataTableToolbarInteractiveDemo />

```tsx title="components/demos/workflows-demos.tsx"
import { DataTableToolbar } from "@formaui/components";

export function CustomersToolbar() {
  return (
    <DataTableToolbar
      resultCount={24}
      onQueryChange={(value) => console.log(value)}
      onRefresh={() => console.log("refresh")}
      onAdd={() => console.log("add")}
      addLabel="Add customer"
    />
  );
}
```

## API Reference

### `DataTableToolbar` Props

| Prop               | 类型                      | 默认值             | 说明             |
| ------------------ | ------------------------- | ------------------ | ---------------- |
| `value`            | `string`                  | `-`                | 受控查询值       |
| `defaultValue`     | `string`                  | `-`                | 非受控初始查询值 |
| `resultCount`      | `number`                  | `-`                | 结果数量         |
| `onQueryChange`    | `(value: string) => void` | `-`                | 查询变化回调     |
| `onRefresh`        | `() => void`              | `-`                | 刷新动作回调     |
| `onAdd`            | `() => void`              | `-`                | 新增动作回调     |
| `queryPlaceholder` | `string`                  | `"Search rows..."` | 查询输入占位文案 |
| `addLabel`         | `string`                  | `"Add row"`        | 新增按钮文案     |
| `className`        | `string`                  | `-`                | 追加容器样式     |

## 可访问性

- 查询输入框包含语义化标签与 `aria-label`。
- 结果数量区域使用 `aria-live="polite"` 提示变化。
- 操作按钮建议保持稳定顺序（刷新在前、新增在后）。
