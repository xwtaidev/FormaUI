# Theme Switcher

`ThemeSwitcher` 用于在浅色与深色主题之间切换，适合作为页面头部的全局控制器。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[180px] items-center justify-center p-6">
    <ThemeSwitcherInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/workflows-demos.tsx"
import * as React from "react";
import { ThemeSwitcher } from "@formaui/components";

export function ThemeControl() {
  const [theme, setTheme] = React.useState<"light" | "dark">("light");

  return (
    <ThemeSwitcher
      theme={theme}
      onToggle={() =>
        setTheme((current) => (current === "dark" ? "light" : "dark"))
      }
    />
  );
}
```

## API Reference

### `ThemeSwitcher` Props

| Prop       | 类型                | 默认值 | 说明                 |
| ---------- | ------------------- | ------ | -------------------- |
| `theme`    | `"light" \| "dark"` | `-`    | 当前主题（必填）     |
| `onToggle` | `() => void`        | `-`    | 触发切换回调（必填） |

## 可访问性

- 切换按钮内置 `aria-label="toggle theme"`。
- 建议在切换后同步更新页面主题变量与持久化状态。
- 可在外层补充当前主题文本提示，提升状态可见性。
