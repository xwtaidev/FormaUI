# Metric Card

`MetricCard` 用于展示核心指标值，并可附带变化趋势和描述信息。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[240px] items-center justify-center p-6">
    <MetricCardInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/workflows-demos.tsx"
import { MetricCard } from "@formaui/components";

export function RevenueMetric() {
  return (
    <MetricCard
      title="Revenue"
      value="$128,430"
      delta="+12.4%"
      description="Compared with last month"
    />
  );
}
```

## API Reference

### `MetricCard` Props

| Prop          | 类型     | 默认值 | 说明             |
| ------------- | -------- | ------ | ---------------- |
| `title`       | `string` | `-`    | 指标标题（必填） |
| `value`       | `string` | `-`    | 指标数值（必填） |
| `delta`       | `string` | `-`    | 趋势变化文案     |
| `description` | `string` | `-`    | 补充说明文案     |

## 可访问性

- 建议标题与数值保持简短，便于快速扫描。
- `delta` 建议包含方向语义（如 `+12%` / `-4%`）。
- 多卡片并列时建议搭配分组标题，提升上下文可读性。
