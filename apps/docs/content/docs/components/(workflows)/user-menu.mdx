# User Menu

`UserMenu` 将头像入口与下拉菜单组合为统一账户操作区，适合导航栏与侧边栏头部。

## 可视化展示

  <div className="flex min-h-[180px] items-center justify-center p-6">
    <UserMenuInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/workflows-demos.tsx"
import { UserMenu } from "@formaui/components";

export function AccountMenu() {
  return (
    <UserMenu
      name="Jane Doe"
      email="jane.doe@formaui.dev"
      onSignOut={() => console.log("sign out")}
    />
  );
}
```

## API Reference

### `UserMenu` Props

| Prop          | 类型         | 默认值 | 说明             |
| ------------- | ------------ | ------ | ---------------- |
| `name`        | `string`     | `-`    | 用户名（必填）   |
| `email`       | `string`     | `-`    | 用户邮箱（必填） |
| `onSignOut`   | `() => void` | `-`    | 退出动作回调     |
| `defaultOpen` | `boolean`    | `-`    | 菜单初始展开状态 |

## 可访问性

- 头像触发器提供 `aria-label="Open user menu"`。
- 下拉内容包含账户名和邮箱，便于读屏理解当前身份。
- 退出动作建议保留清晰文案并配合二次确认策略（按业务需要）。
