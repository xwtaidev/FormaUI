# Filter Bar

`FilterBar` 将查询输入、状态单选和时间区间组合为统一筛选面板，适合数据看板与列表页。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[340px] items-center justify-center p-6">
    <FilterBarInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/workflows-demos.tsx"
import { FilterBar } from "@formaui/components";

export function IssuesFilter() {
  return (
    <FilterBar
      onChange={(value) => console.log(value)}
      statusOptions={[
        { label: "All", value: "all" },
        { label: "Open", value: "open" },
        { label: "Closed", value: "closed" },
      ]}
    />
  );
}
```

## API Reference

### `FilterBar` Props

| Prop               | 类型                              | 默认值                              | 说明             |
| ------------------ | --------------------------------- | ----------------------------------- | ---------------- |
| `value`            | `FilterBarValueInput`             | `-`                                 | 受控筛选值       |
| `defaultValue`     | `FilterBarValueInput`             | `-`                                 | 非受控初始筛选值 |
| `statusOptions`    | `FilterOption[]`                  | `All/Open/Closed`                   | 状态选项列表     |
| `onChange`         | `(value: FilterBarValue) => void` | `-`                                 | 筛选值变化回调   |
| `onReset`          | `() => void`                      | `-`                                 | 重置动作回调     |
| `queryPlaceholder` | `string`                          | `"Search by keyword, owner, or ID"` | 查询占位文案     |
| `className`        | `string`                          | `-`                                 | 追加容器样式     |

### 相关类型

`FilterOption`:

| 字段    | 类型     | 说明     |
| ------- | -------- | -------- |
| `label` | `string` | 选项文案 |
| `value` | `string` | 选项值   |

`FilterBarValue`:

| 字段     | 类型             | 说明       |
| -------- | ---------------- | ---------- |
| `query`  | `string`         | 查询关键词 |
| `status` | `string`         | 状态值     |
| `range`  | `DateRangeValue` | 日期区间   |

## 可访问性

- 查询输入、状态组选项与重置按钮均可键盘访问。
- 状态切换使用 `RadioGroup` 语义，适合单选筛选表达。
- 建议在筛选结果区域同步反馈总数，提升状态可见性。
