# Search Command

`SearchCommand` 用于快速检索并执行动作，支持关键词过滤、描述信息和快捷键提示。

  <div className="flex min-h-[320px] items-center justify-center p-6">
    <SearchCommandInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/workflows-demos.tsx"
import { SearchCommand } from "@formaui/components";

export function CommandPalette() {
  return (
    <SearchCommand
      items={[
        { id: "deploy", label: "Deploy project", shortcut: "D" },
        { id: "rollback", label: "Rollback release", shortcut: "R" },
      ]}
      onSelect={(item) => console.log(item.id)}
    />
  );
}
```

## API Reference

### `SearchCommand` Props

| Prop           | 类型                                | 默认值                    | 说明               |
| -------------- | ----------------------------------- | ------------------------- | ------------------ |
| `items`        | `SearchCommandItem[]`               | `-`                       | 命令项列表（必填） |
| `onSelect`     | `(item: SearchCommandItem) => void` | `-`                       | 选择命令回调       |
| `title`        | `string`                            | `"Command menu"`          | 面板标题           |
| `placeholder`  | `string`                            | `"Search command"`        | 查询输入占位文案   |
| `emptyMessage` | `string`                            | `"No matching commands."` | 无匹配提示文案     |
| `className`    | `string`                            | `-`                       | 追加容器样式       |

### `SearchCommandItem` 结构

| 字段          | 类型       | 默认值 | 说明           |
| ------------- | ---------- | ------ | -------------- |
| `id`          | `string`   | `-`    | 命令唯一标识   |
| `label`       | `string`   | `-`    | 命令标题       |
| `description` | `string`   | `-`    | 命令描述       |
| `keywords`    | `string[]` | `-`    | 额外搜索关键词 |
| `shortcut`    | `string`   | `-`    | 快捷键提示     |

## 可访问性

- 输入框具备 `aria-label`，支持键盘快速筛选。
- 命令项使用按钮语义，适合回车/空格触发。
- 建议 `label` 简洁明确，`shortcut` 与实际键位保持一致。
