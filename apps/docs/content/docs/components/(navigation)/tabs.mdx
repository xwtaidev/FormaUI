# Tabs

`Tabs` 用于在同一页面容器内切换不同内容面板，适合概览/配置/日志等并列信息结构。

## 可视化展示

  <div className="flex min-h-[260px] items-center justify-center p-6">
    <TabsInteractiveDemo />

</div>

```tsx title="components/demos/navigation-core-demos.tsx"
import { Tabs, TabsContent, TabsList, TabsTrigger } from "@formaui/components";

export function ProjectTabs() {
  return (
    <Tabs defaultValue="overview">
      <TabsList>
        <TabsTrigger value="overview">Overview</TabsTrigger>
        <TabsTrigger value="activity">Activity</TabsTrigger>
      </TabsList>
      <TabsContent value="overview">Overview Content</TabsContent>
      <TabsContent value="activity">Activity Content</TabsContent>
    </Tabs>
  );
}
```

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add tabs
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { Tabs, TabsContent, TabsList, TabsTrigger } from "@formaui/components";

export function BasicTabs() {
  return (
    <Tabs defaultValue="profile" className="w-full max-w-md">
      <TabsList>
        <TabsTrigger value="profile">Profile</TabsTrigger>
        <TabsTrigger value="security">Security</TabsTrigger>
      </TabsList>
      <TabsContent value="profile">Profile Settings</TabsContent>
      <TabsContent value="security">Security Settings</TabsContent>
    </Tabs>
  );
}
```

## API Reference

### `Tabs`（Root）Props

`Tabs` 继承 `React.ComponentPropsWithoutRef<typeof TabsPrimitive.Root>`：

| Prop             | 类型                         | 默认值         | 说明             |
| ---------------- | ---------------------------- | -------------- | ---------------- |
| `value`          | `string`                     | `-`            | 受控激活面板值   |
| `defaultValue`   | `string`                     | `-`            | 非受控初始面板值 |
| `onValueChange`  | `(value: string) => void`    | `-`            | 面板变化回调     |
| `orientation`    | `"horizontal" \| "vertical"` | `"horizontal"` | 布局方向         |
| `activationMode` | `"automatic" \| "manual"`    | `"automatic"`  | 激活模式         |
| `className`      | `string`                     | `-`            | 根容器样式扩展   |

### 常用子组件 Props

| 组件          | 关键 Props          | 说明           |
| ------------- | ------------------- | -------------- |
| `TabsList`    | `className`         | 标签触发器容器 |
| `TabsTrigger` | `value`, `disabled` | 标签触发项     |
| `TabsContent` | `value`             | 对应内容面板   |

## 可访问性

- 基于 Radix Tabs，支持键盘在标签间切换。
- `TabsTrigger` 与 `TabsContent` 自动建立语义关联。
- 标签文案应简短且互斥，避免理解歧义。
- 内容区域建议保留稳定高度，减少切换抖动。
