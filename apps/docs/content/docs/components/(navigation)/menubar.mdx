# Menubar

`Menubar` 用于承载常驻命令入口，常见于编辑器、后台工作台等高频操作场景。

它基于 Radix Menubar 构建，支持子菜单、勾选项、禁用项与键盘导航。

## API Reference

### `Menubar`（Root）Props

`Menubar` 继承 `React.ComponentPropsWithoutRef<typeof MenubarPrimitive.Root>`，常用参数：

| Prop            | 类型                      | 默认值 | 说明             |
| --------------- | ------------------------- | ------ | ---------------- |
| `value`         | `string`                  | `-`    | 受控激活菜单值   |
| `defaultValue`  | `string`                  | `-`    | 非受控初始菜单值 |
| `onValueChange` | `(value: string) => void` | `-`    | 菜单切换回调     |
| `className`     | `string`                  | `-`    | 根容器样式扩展   |

### 常用子组件 Props

| 组件                  | 关键 Props                         | 说明           |
| --------------------- | ---------------------------------- | -------------- |
| `MenubarTrigger`      | `disabled`, `className`            | 顶层菜单触发器 |
| `MenubarContent`      | `align`, `sideOffset`, `className` | 下拉内容区     |
| `MenubarItem`         | `disabled`, `inset`                | 普通命令项     |
| `MenubarCheckboxItem` | `checked`, `onCheckedChange`       | 可勾选命令项   |
| `MenubarSubTrigger`   | `inset`                            | 子菜单触发项   |

## 可访问性

- 基于 Radix，支持键盘遍历与子菜单语义。
- 勾选项、禁用项状态对辅助技术可感知。
- 适合常驻命令面板，不适合纯导航信息架构。
