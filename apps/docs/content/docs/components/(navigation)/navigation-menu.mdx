# Navigation Menu

`NavigationMenu` 用于网站或控制台顶部导航，适合承载一级分类入口与二级内容面板。

它基于 Radix Navigation Menu 构建，提供触发器、内容区、指示器和视口等组合能力。

## API Reference

### `NavigationMenu`（Root）Props

`NavigationMenu` 继承 `React.ComponentPropsWithoutRef<typeof NavigationMenuPrimitive.Root>`，常用参数：

| Prop                | 类型                      | 默认值              | 说明             |
| ------------------- | ------------------------- | ------------------- | ---------------- |
| `value`             | `string`                  | `-`                 | 受控激活项值     |
| `defaultValue`      | `string`                  | `-`                 | 非受控初始激活项 |
| `onValueChange`     | `(value: string) => void` | `-`                 | 激活项变化回调   |
| `delayDuration`     | `number`                  | `200`（Radix 默认） | 打开延迟         |
| `skipDelayDuration` | `number`                  | `300`（Radix 默认） | 跳过延迟时长     |
| `className`         | `string`                  | `-`                 | 根节点样式扩展   |

### 常用子组件 Props

| 组件                      | 关键 Props              | 说明                 |
| ------------------------- | ----------------------- | -------------------- |
| `NavigationMenuTrigger`   | `disabled`, `className` | 导航触发按钮         |
| `NavigationMenuContent`   | `className`             | 展开内容面板         |
| `NavigationMenuLink`      | `href`, `className`     | 面板内导航链接       |
| `NavigationMenuIndicator` | `className`             | 当前激活项下方指示器 |
| `NavigationMenuViewport`  | `className`             | 内容区域视口容器     |

### 继承属性

- `NavigationMenuTrigger` 继承 `React.ComponentPropsWithoutRef<typeof NavigationMenuPrimitive.Trigger>`
- `NavigationMenuContent` 继承 `React.ComponentPropsWithoutRef<typeof NavigationMenuPrimitive.Content>`
- `NavigationMenuLink` 继承 `React.ComponentPropsWithoutRef<typeof NavigationMenuPrimitive.Link>`

## 可访问性

- 基于 Radix roving focus，支持键盘在菜单项间移动。
- 触发器与内容区自动暴露展开/收起语义状态。
- 建议每个一级入口保持信息架构清晰，避免超深层级。
- 若用于纯命令操作（如 File/Edit/View），优先考虑 `Menubar`。
