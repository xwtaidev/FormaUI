# DropdownMenu

`DropdownMenu` 用于在按钮附近弹出动作列表，适合列表行操作、卡片快捷操作、用户操作面板等场景。

它支持分组、分隔线、勾选项和子菜单，能在保持界面简洁的同时提供完整操作入口。

## 使用方式

```tsx title="Basic Usage"
import {
  Button,
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuTrigger,
} from "@formaui/components";

export function RowMenu() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="ghost" size="sm">
          Actions
        </Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent align="end">
        <DropdownMenuItem>Edit</DropdownMenuItem>
        <DropdownMenuItem>Duplicate</DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  );
}
```

## API Reference

### `DropdownMenu`（Root）Props

`DropdownMenu` 继承 `React.ComponentPropsWithoutRef<typeof DropdownMenuPrimitive.Root>`：

| Prop           | 类型                      | 默认值               | 说明             |
| -------------- | ------------------------- | -------------------- | ---------------- |
| `open`         | `boolean`                 | `-`                  | 受控开关状态     |
| `defaultOpen`  | `boolean`                 | `false`              | 非受控初始状态   |
| `onOpenChange` | `(open: boolean) => void` | `-`                  | 开关变化回调     |
| `modal`        | `boolean`                 | `true`（Radix 默认） | 是否模态捕获焦点 |

### 常用子组件 Props

| 组件                       | 关键 Props                         | 说明         |
| -------------------------- | ---------------------------------- | ------------ |
| `DropdownMenuTrigger`      | `asChild`                          | 触发元素包装 |
| `DropdownMenuContent`      | `sideOffset`, `align`, `className` | 菜单内容容器 |
| `DropdownMenuItem`         | `disabled`, `inset`                | 普通操作项   |
| `DropdownMenuCheckboxItem` | `checked`, `onCheckedChange`       | 勾选项       |
| `DropdownMenuSubTrigger`   | `inset`                            | 子菜单触发项 |

## 可访问性

- 基于 Radix DropdownMenu，支持键盘导航与焦点管理。
- `DropdownMenuTrigger` 建议使用可访问名称明确的按钮。
- 对危险动作建议增加视觉分隔与二次确认流程。
- 行内操作场景建议使用 `align=\"end\"` 降低遮挡风险。
