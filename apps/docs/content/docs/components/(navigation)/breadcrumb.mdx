# Breadcrumb

`Breadcrumb` 用于展示层级路径并帮助用户快速定位当前位置，常见于后台系统与文档站。

## API Reference

### 组件 Props

| 组件                  | 关键 Props                | 说明                    |
| --------------------- | ------------------------- | ----------------------- |
| `Breadcrumb`          | `aria-label`, `className` | 外层导航容器（`<nav>`） |
| `BreadcrumbList`      | `className`               | 路径列表容器（`<ol>`）  |
| `BreadcrumbItem`      | `className`               | 路径节点（`<li>`）      |
| `BreadcrumbLink`      | `href`, `className`       | 可点击路径节点          |
| `BreadcrumbPage`      | `className`               | 当前页节点（不可点击）  |
| `BreadcrumbSeparator` | `children`, `className`   | 分隔符节点              |

### 继承属性

- `Breadcrumb` 继承 `React.ComponentPropsWithoutRef<"nav">`
- `BreadcrumbList` 继承 `React.ComponentPropsWithoutRef<"ol">`
- `BreadcrumbLink` 继承 `React.ComponentPropsWithoutRef<"a">`

## 可访问性

- 使用原生 `<nav aria-label="Breadcrumb">` 语义。
- 当前页通过 `aria-current="page"` 标记。
- 分隔符使用 `aria-hidden`，避免重复朗读。
- 路径层级建议控制在 2~4 层，避免冗长。
