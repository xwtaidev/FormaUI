# Affix

`Affix` 用于在页面滚动过程中固定关键操作区域，适合筛选条、快捷操作栏、浮动提示条等场景。

## 可视化展示

<div className="not-prose my-4 overflow-hidden rounded-xl border border-fd-border">
  <div className="flex min-h-[340px] items-center justify-center p-6">
    <AffixInteractiveDemo />
  </div>
</div>

```tsx title="components/demos/navigation-utility-demos.tsx"
import { Affix, Button } from "@formaui/components";

export function ToolbarAffix() {
  return (
    <Affix offsetTop={12}>
      <Button size="sm">Quick Action</Button>
    </Affix>
  );
}
```

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add affix
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/components
```

## 使用方式

```tsx title="Basic Usage"
import { Affix, Button } from "@formaui/components";

export function FloatingFilterBar() {
  return (
    <Affix offsetTop={8}>
      <Button variant="outline" size="sm">
        筛选条件
      </Button>
    </Affix>
  );
}
```

## API Reference

### Props

| Prop           | 类型                         | 默认值 | 说明             |
| -------------- | ---------------------------- | ------ | ---------------- |
| `offsetTop`    | `number`                     | `0`    | 距离顶部偏移     |
| `offsetBottom` | `number`                     | `-`    | 距离底部偏移     |
| `onChange`     | `(affixed: boolean) => void` | `-`    | 固定状态变化回调 |
| `className`    | `string`                     | `-`    | 容器样式扩展     |
| `style`        | `React.CSSProperties`        | `-`    | 内联样式扩展     |

## 可访问性

- `Affix` 不改变子元素语义，仅处理布局行为。
- 建议固定区域内控件保持清晰可读的可访问名称。
- 固定后仍应保持与正文合理间距，避免遮挡内容。
