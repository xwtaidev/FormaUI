# FormaUI v0.7 Release Notes

## Snapshot

- Date: 2026-04-30
- Status: GA Ready
- Version Line: `v0.7.1 -> v0.7.6`

## TL;DR

v0.7 完成 16 个 P0 基础组件的端到端交付（实现、registry、docs、测试、CLI discoverability、示例覆盖）。

## Highlights

- 16 个组件全量完成：实现/注册/文档/测试全部 `done`。
- CLI `list/search/info/add` 能检索与安装全部 v0.7 新组件。
- `examples/next-app` 增加 v0.7 组合演示并通过构建门禁。

## Delivered Scope

- Wave v0.7.2：`cascader`、`tree-select`、`transfer`、`time-picker`、`color-picker`、`rate`
- Wave v0.7.3：`descriptions`、`result`、`timeline`、`segmented`、`spin`、`image`
- Wave v0.7.4：`affix`、`anchor`、`backtop`、`tree`

## Upgrade Notes

- 对外无破坏性 API 变更。
- 在 Next.js App Router 中组合交互组件时，建议显式使用客户端边界（`"use client"`）。

## Quality Gates

执行日期：2026-04-30 (Asia/Shanghai)

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```

结果：全部通过（存在非阻断第三方 `"use client"` 构建警告）。

## Known Limitations

- 官方框架支持仍以 React 为主。
- 私有 registry 权限与鉴权能力不在本版本范围。
