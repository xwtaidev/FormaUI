# FormaUI v0.1 Release Notes

## Snapshot

- Date: 2026-04-25
- Status: GA
- Release Tag: `v0.1`

## TL;DR

v0.1 是 FormaUI 首个完整版本，打通了 Source-Owned 设计系统的端到端工作流：组件、区块、模板、主题、registry、CLI、文档与示例全链路可用。

## Highlights

- 设计基础交付：typed tokens + `default/avocado` 主题 + light/dark。
- 组件层交付：13 primitives + 8 composites。
- 产品层交付：8 个 SaaS/AI blocks + `ai-console-lite` 模板。
- 工具链交付：`init/add/block add/template add/theme add`。

## Delivered Scope

- Registry 资产：22 组件、8 blocks、2 themes、1 template。
- Docs 覆盖安装、快速开始、组件、区块、CLI、registry。
- 官方 Next.js 示例可构建并演示完整资产链。

## Upgrade Notes

- v0.1 为起始 GA 线，无历史迁移要求。
- 建议直接以 `init -> add -> doctor` 建立项目基线。

## Quality Gates

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
pnpm registry:build
```

结果：全部通过。

## Known Limitations

- 官方支持以 React 为主。
- 私有 registry、Figma Kit、Studio 能力不在 v0.1 范围。
