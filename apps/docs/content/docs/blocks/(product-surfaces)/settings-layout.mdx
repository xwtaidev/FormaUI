# SettingsLayout

`SettingsLayout` 是面向产品设置页的标准区块，帮助你快速交付具备一致交互模式的设置中心。

它基于 `Tabs + Card + Input + Switch` 组合实现，覆盖常见的资料、通知与安全配置场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add settings-layout
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { SettingsLayout } from "@formaui/blocks";

export function SettingsPage() {
  return <SettingsLayout />;
}
```

## 案例

### 1. 作为账户设置页主内容

```tsx title="Settings Center"
import { SettingsLayout } from "@formaui/blocks";

export function WorkspaceSettings() {
  return (
    <section className="mx-auto max-w-4xl">
      <SettingsLayout />
    </section>
  );
}
```

## API Reference

### Props

当前版本 `SettingsLayout` 不暴露组件 props，默认渲染预设设置结构。

如需接入真实配置项，建议安装后在 `components/settings-layout.tsx` 中替换默认字段与开关逻辑。

### 依赖组件

- `tabs`
- `card`
- `input`
- `switch`

## 响应式与可访问性

- 标签页结构在小屏下仍保持可触达，适配移动端滚动。
- 使用语义化表单控件（`input` / `switch`），默认支持键盘焦点和读屏朗读。
