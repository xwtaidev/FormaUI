# ApiKeyManager

`ApiKeyManager` 用于展示和管理工作区 API Key 生命周期，适合接入开发者后台或安全设置页。

它通过 `Card + Badge + Button` 组合呈现密钥信息、状态与操作入口。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add api-key-manager
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { ApiKeyManager } from "@formaui/blocks";

export function ApiAccessPage() {
  return <ApiKeyManager />;
}
```

## 案例

### 1. 嵌入安全设置区域

```tsx title="Security Section"
import { ApiKeyManager, SettingsLayout } from "@formaui/blocks";

export function SecurityAndKeys() {
  return (
    <div className="space-y-6">
      <SettingsLayout />
      <ApiKeyManager />
    </div>
  );
}
```

## API Reference

### Props

当前版本 `ApiKeyManager` 不暴露组件 props，默认渲染演示密钥数据。

接入生产数据时，建议安装后在 `components/api-key-manager.tsx` 中将 `demoApiKeys` 替换为你的密钥列表与操作逻辑。

### 依赖组件

- `card`
- `badge`
- `button`

## 响应式与可访问性

- 密钥项在窄屏下自动换行，状态与操作区保持可触达。
- 状态文本与标签并存（`active/revoked`），避免仅依赖颜色传达状态。
