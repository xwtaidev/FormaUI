# BillingPanel

`BillingPanel` 用于承载订阅计划、席位使用和账单动作，是 SaaS 后台常见的计费管理模块。

它通过 `Card + Badge + Button` 组合，提供结构化账单信息与关键动作入口。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add billing-panel
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { BillingPanel } from "@formaui/blocks";

export function BillingPage() {
  return <BillingPanel />;
}
```

## 案例

### 1. 与成员管理组合成运营页

```tsx title="Billing + Team"
import { BillingPanel, TeamMembersTable } from "@formaui/blocks";

export function WorkspaceOps() {
  return (
    <div className="space-y-4">

```

## API Reference

### Props

当前版本 `BillingPanel` 不暴露组件 props，默认渲染演示订阅信息。

如需对接真实账单系统，建议安装后在 `components/billing-panel.tsx` 中替换价格、席位和按钮行为。

### 依赖组件

- `card`
- `badge`
- `button`

## 响应式与可访问性

- 账单摘要卡片在小屏单列、桌面多列，信息分层清晰。
- 关键动作按钮具备明确文案，便于键盘与辅助技术识别。
