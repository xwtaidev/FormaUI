# NotificationPanel

`NotificationPanel` 用于聚合工作区事件与告警，帮助团队快速完成分级处理与操作分派。

它内置 `SearchCommand` 快捷动作区与通知列表，适合作为运营控制台的告警入口。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add notification-panel
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { NotificationPanel } from "@formaui/blocks";

export function AlertsPage() {
  return <NotificationPanel />;
}
```

## 案例

### 1. 自定义文案并接入业务通知

```tsx title="Custom Notifications"
import { NotificationPanel } from "@formaui/blocks";

const notifications = [
  {
    id: "n1",
    title: "Usage threshold reached",
    summary: "Workspace token usage exceeded 80%.",
    level: "critical",
    time: "2 min ago"
  }
] as const;

export function IncidentInbox() {
  return (
    <NotificationPanel
      title="Incident inbox"
      description="Critical signals across AI workloads and billing operations."
      notifications={[...notifications]}
    />
  );
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `notifications` | `DemoNotification[]` | `demoNotifications` | 通知数据列表 |
| `title` | `string` | `"Notifications"` | 区块标题 |
| `description` | `string` | `"Critical updates and workspace events from the last 24 hours."` | 区块说明 |

`DemoNotification` 结构：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | `string` | 通知唯一标识 |
| `title` | `string` | 通知标题 |
| `summary` | `string` | 通知摘要 |
| `level` | `"critical" \| "info" \| "success"` | 告警级别 |
| `time` | `string` | 时间信息 |

### 依赖组件

- `card`
- `badge`
- `button`
- `search-command`

## 响应式与可访问性

- 通知项在窄屏纵向堆叠，较宽屏幕自动切换为行式布局。
- 告警级别使用文本标签（`critical/info/success`）并配合样式增强识别。
