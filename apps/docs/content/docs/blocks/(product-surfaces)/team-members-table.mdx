# TeamMembersTable

## Overview

`TeamMembersTable` combines card framing with `DataTable` to render sortable member directories.

## Install Command

```bash
npx formaui block add team-members-table
```

## Included Composition

- Card shell (`card`)
- Status tags (`badge`)
- Sortable data rows (`data-table`)

## Use Case

Use in admin and workspace settings pages for ownership and escalation matrices.
