# TeamMembersTable

`TeamMembersTable` 用于展示团队成员目录，支持角色、团队、地点与状态等信息的结构化呈现。

它基于 `Card + DataTable + Badge` 组合，适合权限管理、成员治理与升级流程场景。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add team-members-table
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { TeamMembersTable } from "@formaui/blocks";

export function MembersPage() {
  return <TeamMembersTable />;
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `members` | `DemoTeamMember[]` | `demoTeamMembers` | 成员数据列表 |
| `title` | `string` | `"Team members"` | 区块标题 |
| `description` | `string` | `"Directory of owners across product, engineering, and operations."` | 区块说明 |

`DemoTeamMember` 结构：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | `string` | 成员唯一标识 |
| `name` | `string` | 成员姓名 |
| `role` | `string` | 职责角色 |
| `team` | `string` | 所属团队 |
| `location` | `string` | 工作地点 |
| `status` | `"active" \| "away" \| "offline"` | 在线状态 |

### 依赖组件

- `card`
- `badge`
- `data-table`

## 响应式与可访问性

- 表格容器遵循 DataTable 的响应式策略，适配桌面与窄屏阅读。
- 状态通过文本与标签共同表达，提升可读性与无障碍友好度。
