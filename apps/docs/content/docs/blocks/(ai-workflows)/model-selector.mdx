# ModelSelector

## Overview

`ModelSelector` helps teams compare model options by provider, latency, cost, and context limits before routing traffic.

## Install Command

```bash
npx formaui block add model-selector
```

## Included Composition

- Selection container (`card`)
- Status tags (`badge`)
- Primary action (`button`)
- Provider filtering (`select`)

## Semantic Token Usage

This block consumes semantic aliases such as:

- `--semantic-model-chip-bg`
- `--semantic-model-chip-selected-bg`
- `--semantic-text-primary`
- `--semantic-border-default`
