# TokenUsageChart

`TokenUsageChart` 是一个轻量的用量可视化区块，用于展示最近时间窗口内的 Token 消耗趋势。

它基于 `Card + Badge` 组合实现，适合作为 AI 产品仪表盘中的基础监控卡片。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add token-usage-chart
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { TokenUsageChart } from "@formaui/blocks";

export function UsagePanel() {
  return (
    <section className="grid gap-4 md:grid-cols-2">
      <TokenUsageChart />
    </section>
  );
}
```

## API Reference

### Props

当前版本 `TokenUsageChart` 不暴露组件 props，直接渲染预设结构与演示数据。

如果需要接入真实业务数据，建议安装后在你的仓库内修改 `components/token-usage-chart.tsx`，将 `demoTokenUsage` 替换为后端指标数据。

### 依赖组件

- `card`
- `badge`

## 响应式与可访问性

- 默认可在单列或双列网格中自然伸缩。
- 条形图使用文本标签同步呈现数值，不依赖纯颜色传达信息。
- 卡片标题与描述提供语义化信息，便于快速扫描与读屏理解。
