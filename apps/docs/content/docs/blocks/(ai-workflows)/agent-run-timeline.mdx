# AgentRunTimeline

`AgentRunTimeline` 用于展示近期 Agent 运行记录，帮助团队快速感知执行状态、失败任务与当前运行中的作业。

它由 `Card + Badge + Button` 组合构成，适合 AI 控制台中的运行监控面板。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add agent-run-timeline
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { AgentRunTimeline } from "@formaui/blocks";

export function RunsPanel() {
  return <AgentRunTimeline />;
}
```

## API Reference

### Props

当前版本 `AgentRunTimeline` 不暴露组件 props，默认渲染演示运行记录。

接入线上数据时，建议安装后在 `components/agent-run-timeline.tsx` 中将 `demoAgentRuns` 替换为你的任务执行数据源。

### 依赖组件

- `card`
- `badge`
- `button`

## 响应式与可访问性

- 记录条目在窄屏下自动换行，宽屏保持信息并排。
- 状态标签使用 `Badge` 文本表达（`success/running/failed`），避免仅凭颜色识别。
- 头部操作按钮使用语义化 `<button>`，支持键盘触发。
