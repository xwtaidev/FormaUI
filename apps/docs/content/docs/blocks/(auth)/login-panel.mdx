# LoginPanel

`LoginPanel` 是用于账号登录的认证入口区块，适合产品后台、管理系统和工作区登录页。

它基于 `Card + Input + Checkbox + Button` 组合实现，内置邮箱、密码与记住登录状态字段。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add login-panel
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { LoginPanel } from "@formaui/blocks";

export function SignInPage() {
  return (
    <main className="flex min-h-screen items-center justify-center p-6">
      <LoginPanel />
    </main>
  );
}
```

## API Reference

### Props

当前版本 `LoginPanel` 不暴露组件 props，默认渲染标准登录表单结构。

如需接入真实认证流程，建议安装后在 `components/login-panel.tsx` 中绑定提交事件、校验与 API 请求逻辑。

### 依赖组件

- `card`
- `input`
- `checkbox`
- `button`

## 响应式与可访问性

- 表单默认容器宽度为 `max-w-md`，在移动端和桌面端都保持良好可读性。
- 字段标签通过 `htmlFor` 关联输入框，提升读屏可用性。
- 使用语义化表单控件，支持键盘聚焦与操作。
