# LogoCloud

`LogoCloud` 用于展示生态兼容与技术栈背书，常用于 Hero 后的信任增强区。

它以轻量网格展示框架或合作方名称，默认内置常见前端技术栈。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add logo-cloud
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { LogoCloud } from "@formaui/blocks";

export function TrustSection() {
  return <LogoCloud />;
}
```

## 案例

### 1. 自定义展示品牌

```tsx title="Custom Stack Logos"
import { LogoCloud } from "@formaui/blocks";

export function EcosystemProof() {
  return (
    <LogoCloud
      title="Integrates with your existing workflow"
      description="Adopt FormaUI without replacing your current stack."
      logos={["Next.js", "React", "TypeScript", "Node.js", "Supabase", "Stripe"]}
    />
  );
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `title` | `string` | `"Works with your stack"` | 区块标题 |
| `description` | `string` | 默认描述文案 | 区块描述 |
| `logos` | `ReadonlyArray<string>` | 内置技术栈数组 | 展示项列表 |

### 依赖组件

- `card`

## 响应式与可访问性

- Logo 项以文本形式展示，低对比环境仍可识别。
- 默认网格在移动端 2 列，桌面端 3 列。
