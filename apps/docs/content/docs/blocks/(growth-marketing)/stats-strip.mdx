# StatsStrip

`StatsStrip` 用于展示关键业务指标，常用于营销页中的社会证明和转化增强区。

它通过卡片化数字与标签让用户快速感知产品规模与成熟度。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add stats-strip
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { StatsStrip } from "@formaui/blocks";

export function LandingStats() {
  return <StatsStrip />;
}
```

## 案例

### 1. 自定义关键指标

```tsx title="Custom Metrics"
import { StatsStrip } from "@formaui/blocks";

const stats = [
  { label: "Active Workspaces", value: "1,240+" },
  { label: "Automation Runs / Day", value: "82k" },
  { label: "Customer Retention", value: "96%" }
];

export function GrowthProof() {
  return (
    <StatsStrip
      title="Numbers teams trust"
      description="Operational reliability across product and growth workflows."
      stats={stats}
    />
  );
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `title` | `string` | `"Trusted by source-owned product teams"` | 区块标题 |
| `description` | `string` | 默认描述文案 | 区块描述 |
| `stats` | `ReadonlyArray<{ label: string; value: string }>` | 内置 4 项指标 | 指标列表 |

### 依赖组件

- `card`

## 响应式与可访问性

- 指标网格在 `sm/lg` 断点逐步扩列。
- 数值与标签组合表达，避免仅视觉强调而缺乏语义说明。
