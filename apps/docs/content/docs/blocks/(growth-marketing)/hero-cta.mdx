# HeroCta

`HeroCta` 是增长页面的首屏入口区块，用于承载核心价值主张和主次转化动作。

它基于 `Card + Badge + Button` 组合实现，适合产品发布页、活动页与官网落地页。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add hero-cta
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { HeroCta } from "@formaui/blocks";

export function LandingHero() {
  return <HeroCta />;
}
```

## 案例

### 1. 自定义营销文案与链接

```tsx title="Custom Hero"
import { HeroCta } from "@formaui/blocks";

export function CampaignHero() {
  return (
    <HeroCta
      eyebrow="Spring Launch"
      title="Deploy faster with source-owned UI"
      description="Install production-ready blocks and customize directly in your repository."
      primaryCtaLabel="Start free"
      secondaryCtaLabel="Contact sales"
      primaryCtaHref="/docs/quick-start"
      secondaryCtaHref="/contact"
    />
  );
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `eyebrow` | `string` | `"Marketing blocks"` | 标题上方提示文本 |
| `title` | `string` | `"Ship FormaUI faster"` | 主标题 |
| `description` | `string` | 默认描述文案 | 副标题描述 |
| `primaryCtaLabel` | `string` | `"Start building"` | 主按钮文案 |
| `secondaryCtaLabel` | `string` | `"Book a live demo"` | 次按钮文案 |
| `primaryCtaHref` | `string \| undefined` | `undefined` | 主按钮跳转地址 |
| `secondaryCtaHref` | `string \| undefined` | `undefined` | 次按钮跳转地址 |

### 依赖组件

- `card`
- `badge`
- `button`

## 响应式与可访问性

- 主标题在移动端与桌面端自动调整字号。
- CTA 使用语义化按钮，支持键盘交互。
