# FeatureGrid

`FeatureGrid` 用于在营销页面中结构化展示产品功能卖点，帮助用户快速理解能力边界。

它由标题区和卡片网格组成，默认提供 4 个示例特性卡片。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add feature-grid
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { FeatureGrid } from "@formaui/blocks";

export function LandingFeatures() {
  return <FeatureGrid />;
}
```

## 案例

### 1. 自定义卖点列表

```tsx title="Custom Features"
import { FeatureGrid } from "@formaui/blocks";

const features = [
  { title: "Source-owned by default", description: "Keep full code ownership in your repo." },
  { title: "Registry-native install", description: "Ship updates through one registry workflow." }
];

export function ProductHighlights() {
  return (
    <FeatureGrid
      title="Built for modern product teams"
      description="Composable blocks across docs, apps, and marketing surfaces."
      features={features}
    />
  );
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `title` | `string` | `"Everything teams need to ship"` | 区块标题 |
| `description` | `string` | 默认描述文案 | 区块描述 |
| `features` | `ReadonlyArray<{ title: string; description: string }>` | 内置 4 项示例 | 特性列表 |

### 依赖组件

- `card`
- `badge`

## 响应式与可访问性

- 网格在 `md` 及以上为双列，小屏回落单列。
- 每个特性都有标题与描述，便于视觉与读屏理解。
