# FinalCta

`FinalCta` 用于营销页面末尾的最终转化，帮助用户在阅读完成后快速进入下一步动作。

它支持主副 CTA、文案和链接配置，默认采用强调背景样式增强收尾表达。

## 安装方式

### 通过 CLI 安装

```bash title="CLI"
npx formaui add final-cta
```

### 通过包依赖使用

```bash title="Package Manager"
pnpm add @formaui/blocks
```

## 使用方式

```tsx title="Basic Usage"
import { FinalCta } from "@formaui/blocks";

export function LandingFinalSection() {
  return <FinalCta />;
}
```

## 案例

### 1. 定制收尾文案与按钮

```tsx title="Custom Final CTA"
import { FinalCta } from "@formaui/blocks";

export function CampaignFinalCta() {
  return (
    <FinalCta
      eyebrow="Limited time"
      title="Ready to ship your next campaign page?"
      description="Install composable blocks and publish faster without black-box lock-in."
      primaryCtaLabel="Start Quick Start"
      secondaryCtaLabel="Read integration guides"
      primaryCtaHref="/docs/quick-start"
      secondaryCtaHref="/docs/guides/integrations-nextjs"
    />
  );
}
```

## API Reference

### Props

| Prop | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `eyebrow` | `string` | `"Ready when you are"` | 标题上方提示文本 |
| `title` | `string` | 默认标题文案 | 主标题 |
| `description` | `string` | 默认描述文案 | 副标题描述 |
| `primaryCtaLabel` | `string` | `"Open Quick Start"` | 主按钮文案 |
| `secondaryCtaLabel` | `string` | `"Browse Docs"` | 次按钮文案 |
| `primaryCtaHref` | `string \| undefined` | `undefined` | 主按钮跳转地址 |
| `secondaryCtaHref` | `string \| undefined` | `undefined` | 次按钮跳转地址 |

### 依赖组件

- `card`
- `badge`
- `button`

## 响应式与可访问性

- CTA 按钮组支持换行布局，移动端仍可完整点击。
- 语义化按钮文案有助于读屏识别下一步操作。
